# 快速开始指南

## 环境要求

- Claude Code (最新版本)
- Git (用于版本管理)
- 项目目录：`d:\VSCODE\1项目\jubensha`

## 快速创作一个剧本

### 方式一：交互式引导（推荐新手）

```bash
# 1. 进入项目
cd d:\VSCODE\1项目\jubensha

# 2. 一键创建完整剧本
# 系统会引导你逐步输入信息，自动完成整个创作流程
/create-script
```

### 方式二：分步创作（推荐有经验的创作者）

```bash
# 1. 初始化新剧本项目
/init-script "剧本标题" "类型" "人数"

# 2. 头脑风暴创意
/brainstorm

# 3. 设计世界观
/design-world

# 4. 设计剧情架构
/design-plot

# 5. 创建角色
/create-characters

# 6. 设计线索系统
/design-clues

# 7. 撰写剧本
/write-script

# 8. 审核剧本
/review-script

# 9. 生成 DM 指南
/generate-dm-guide

# 10. 导出成品
/export-script
```

### 方式三：指定参数快速创建

```bash
# 直接指定关键信息
/create-script "黄昏之馆" "本格推理" "6人" "进阶"
```

## 创作示例

以下是一个 6 人本格推理本的创作过程示例：

### 剧本信息
- **名称**：黄昏之馆
- **类型**：本格推理
- **人数**：6人（3男3女）
- **难度**：进阶
- **时长**：3.5-4小时

### 创作流程时间参考

| 阶段 | 预估时间 | 说明 |
|------|---------|------|
| 创意构思 | 5-10分钟 | 头脑风暴核心诡计 |
| 世界观构建 | 10-15分钟 | 设定时代背景和场景 |
| 剧情架构 | 15-20分钟 | 设计核心诡计和剧情大纲 |
| 角色创建 | 20-30分钟 | 为6个角色创建详细档案 |
| 线索设计 | 15-20分钟 | 设计证据链和推理路径 |
| 剧本撰写 | 30-45分钟 | 撰写完整剧本和角色剧本 |
| 审核优化 | 15-20分钟 | 多维度审核和修改 |
| DM指南 | 10-15分钟 | 生成主持人手册 |
| **总计** | **约2-3小时** | 创作一个完整剧本 |

## 提示与建议

### 新手建议
- 初次使用建议从 `/create-script` 开始，体验完整流程
- 先从"新手"难度开始创作
- 使用 `/brainstorm` 激发创意灵感

### 进阶建议
- 分步使用各个命令，精细控制每个环节
- 在不同的创作阶段之间查看中间产物
- 利用 `/review-script` 的"快速"模式进行阶段性检查

### 多人协作
- 使用 Git 管理剧本版本
- 不同阶段的创作可以在不同分支上进行
- 审核报告作为 Pull Request 的附件
