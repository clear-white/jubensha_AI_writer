## 剧本正文格式

### 分幕格式
```markdown
## 第X幕：[幕名称]

**建议时长：** XX分钟

**主持人口播：**
> （DM朗读给所有玩家的内容）
> 这里是口播正文...

**本幕目标：**
- 玩家需要完成的目标1
- 玩家需要完成的目标2

**关键信息：**
- 本幕释放的关键信息

**本幕流程：**
1. 步骤1描述
2. 步骤2描述
...

---

### 第X幕-[角色名] 私密信息

**本幕新增信息：**
（仅该角色知道的信息）

**本幕任务：**
（该角色需要完成的任务）
```

### 角色剧本格式
```markdown
# [角色名称] 角色剧本

## 角色信息卡
- **姓名：** [姓名]
- **年龄：** [年龄]
- **职业：** [职业]
- **外貌：** [外貌描述]
- **性格：** [性格描述]
- **口头禅：** [口头禅]

## 你的背景故事
（角色的完整背景故事，供玩家阅读）

## 案发当日

### [时间] - [地点]
（叙事正文）
...

## 你的秘密
（仅供你知晓的信息，不可直接告诉其他玩家）

## 你的任务
1. 任务1描述
2. 任务2描述
...

## 你知道的信息
- 信息1
- 信息2
...
```

### 线索卡格式
```markdown
### 线索C-XXX：[线索名称]

**类型：** 物证/人证/书证/场景/时间
**发现者：** [发现该线索的角色/公共]
**发现方式：** [如何发现]

**描述：**
（线索的详细描述）

**可解读信息：**
1. 可能的解读A
2. 可能的解读B
...
```

### DM 指南格式
```markdown
# [剧本名称] - DM 指南

## 基本信息
- **类型：** [类型]
- **人数：** [人数]
- **时长：** [时长]
- **难度：** [难度]

## 游戏流程总览
（各轮次的流程表格）

## 各幕主持指南

### 第X幕：[幕名称]
**准备事项：** ...
**主持话术：** ...
**时间控制：** ...
**关键引导：** ...
**应急预案：** ...
...
```

## 文件命名规范

| 元素 | 命名规则 | 示例 |
|------|---------|------|
| 剧本文件 | 序号-中文名称.md | `1-世界观设定.md` |
| 角色文件 | 4-角色剧本-角色名.md | `4-角色剧本-林婉儿.md` |
| 线索文件 | 线索卡/线索-轮次-编号.md | `线索卡/线索-01-C001.md` |
| 物证文件 | 物证卡片/物证-编号-名称.md | `物证卡片/物证-01-染血匕首.md` |
| 图片资源 | 英文或拼音命名 | `character-portrait-linwaner.png` |

## 格式通用规则

1. 使用 UTF-8 编码
2. 使用 Markdown 格式
3. 每个文件在开头标注最后修改日期
4. 引用其他文件使用相对路径 `[角色档案](4-角色剧本-林婉儿.md)`
5. 敏感/剧透内容使用 `>! spoiler !<` 标记
