# Agent 使用指南

## Agent 总览

本系统包含 10 个专用 Agent，覆盖剧本杀创作的完整流程。

## Agent 详解

### 1. Story Architect（故事架构师）
- **模型**: Claude Opus 4.8
- **用途**: 设计世界观、核心诡计、剧情大纲
- **何时调用**: 项目启动阶段
- **输出**: 世界观设定、剧情大纲、核心诡计文档
- **技巧**: 提供越具体的核心创意方向，输出质量越高

### 2. Character Designer（角色设计师）
- **模型**: Claude Sonnet 5
- **用途**: 创建角色档案、动机、关系网
- **何时调用**: 剧情大纲完成后
- **输出**: 每个角色的完整档案
- **技巧**: 可以分批创建角色（每次2-3个），确保质量

### 3. Plot Weaver（剧情编织者）
- **模型**: Claude Opus 4.8
- **用途**: 编织时间线、事件链、反转
- **何时调用**: 剧情大纲和角色完成后
- **输出**: 时间线、事件链、反转设计
- **技巧**: 需要和前序设计紧密结合，确保一致性

### 4. Clue Designer（线索设计师）
- **模型**: Claude Sonnet 5
- **用途**: 设计线索系统、证据链
- **何时调用**: 时间线和反转完成后
- **输出**: 完整线索库、推理路径
- **技巧**: 从真相反推线索，确保覆盖所有推理路径

### 5. Script Writer（剧本作家）
- **模型**: Claude Sonnet 5
- **用途**: 撰写完整剧本正文
- **何时调用**: 所有设计完成后
- **输出**: 完整剧本、角色独立剧本
- **技巧**: 可以分幕逐步撰写，利于控制质量

### 6. Dialogue Writer（对白作家）
- **模型**: Claude Sonnet 5
- **用途**: 撰写角色对白和NPC对话
- **何时调用**: 剧本撰写中
- **输出**: NPC对话脚本、角色语言风格指南
- **技巧**: 和 Script Writer 配合使用

### 7. Props Designer（道具设计师）
- **模型**: Claude Haiku 4.5
- **用途**: 设计物证、场景道具
- **何时调用**: 剧本基本完成后
- **输出**: 物证卡片、场景布置指南
- **技巧**: 使用 Haiku 模型足够，任务相对独立

### 8. Reviewer（审核编辑）
- **模型**: Claude Opus 4.8
- **用途**: 多维度审核、质量评估
- **何时调用**: 剧本完成后
- **输出**: 审核报告、问题清单
- **技巧**: 用 Opus 模型确保审核深度

### 9. DM Guide Writer（DM指南作家）
- **模型**: Claude Sonnet 5
- **用途**: 编写主持人手册
- **何时调用**: 剧本审核通过后
- **输出**: 完整DM指南
- **技巧**: 确保覆盖所有可能的主持情况

### 10. Worldbuilder（世界观构建师）
- **模型**: Claude Sonnet 5
- **用途**: 详细世界观扩展
- **何时调用**: 创意构思阶段
- **输出**: 详细世界观文档
- **技巧**: 和 Story Architect 配合，先框架后细节

## Agent 调用最佳实践

### 何时使用 Agent 工具
- 跨多步的复杂任务（如"设计一个完整的诡计系统"）
- 需要创造性输出的任务
- 需要在子上下文中独立处理的任务

### 何时直接执行
- 简单的文件读写操作
- 已经明确定义的单一任务
- 审核和检查类任务

### Agent 间的协作模式

```
独立并行：无依赖关系时
  Agent A ──┐
             ├──→ 合成
  Agent B ──┘

流水线：有依赖关系时
  Agent A → Agent B → Agent C

混合：常见的创作模式
  ┌─ Agent A1 ─┐
  │  Agent A2   │──→ Agent C
  │  Agent A3   │
  └─────────────┘
```

## 模型选择策略

| 任务类型 | 推荐模型 | 原因 |
|---------|---------|------|
| 创意构思 | Opus 4.8 | 需要深度思考 |
| 逻辑设计 | Opus 4.8 | 需要严谨推理 |
| 内容撰写 | Sonnet 5 | 性价比最佳 |
| 简单设计 | Haiku 4.5 | 低成本 |
| 审核评估 | Opus 4.8 | 需要批判性思维 |
