### E-01：集结与意识投射【主线·起始】

**事件编号**：E-01
**所属幕次**：第一幕（开场）
**触发条件**：游戏开始，DM引导
**类型**：固定事件

**事件描述**：
六位调查员在DMB总部的DEAS舱体区集合。他们接受了任务简报——陆震霆博士死亡，梦境数据存在异常，需要深入调查。技术组帮助每位调查员确认锚定物状态，随后链接神经接口，进入DEAS舱体。

**进入过程**（30秒的"意识投射"体验）：
1. 躺入舱体，闭合玻璃罩
2. 耳边响起低频的嗡鸣声——DEAS系统启动
3. 视觉从DMB总部的白色灯光逐渐模糊，转为全黑
4. 身体感觉消失——无法感觉到四肢、躯干
5. 然后是一阵强烈的"坠落感"——仿佛从极高的地方自由落体
6. 眼前开始出现模糊的光影——那是陆震霆记忆碎片的"流"
7. 最后——"落地"——脚踏实地的感觉恢复

**关键发现**：无（起始事件，建立沉浸感）
**线索获取**：无
**后续影响**：调查员正式进入梦境

**DM执行要点**：
- 建议用缓和的语速描述进入过程，给予玩家充分的时间"进入角色"
- 强调"坠落感"的强烈程度——暗示DEAS链接的不适感
- 确认每位玩家携带的锚定物，记录在DM笔记中
- 进入后，每位玩家获得一份初始信息卡（关于他们在"梦境中"看到的第一个场景）

### E-02：表层探索与记忆异常发现【主线·引导】

**事件编号**：E-02
**所属幕次**：第一幕
**触发条件**：调查员在观澜山庄自由探索约15-20分钟
**类型**：触发事件（时间触发+行为触发）

**事件描述**：
当调查员在别墅中探索一段时间后——无论他们进入哪个区域——都会观察到陆震霆"记忆化身"的不一致性。这个化身是陆震霆在书房中的行为投影，但它展示出以下异常：

**异常A——服装交替**：
调查员如果持续观察记忆化身超过3分钟，就会注意到它会在灰色丝绸睡衣和蓝色商务西装之间反复切换。切换发生时没有任何过渡——前一秒还是睡衣，后一秒就变成了西装，但动作保持连续。

**异常B——台词偏差**：
记忆化身偶尔会自言自语。但同一个句子出现了两个不同的版本：
- 版本1（约60%概率）："我需要想清楚……"（语气犹疑、疲惫）
- 版本2（约40%概率）："我必须做个了断。"（语气坚定、带着决绝）
这两个版本交替出现，与记忆化身当时的动作无关。

**异常C——时钟停滞**：
别墅中的所有钟表都停在11:47——但这是"陆震霆的死亡时间"，本应一致。然而，如果调查员将书房中的钟表与大厅中的钟表仔细对比，会发现：书房的钟表指针比其他房间的钟表偏离了约2-3度（秒针位置不同）。这说明不同的钟表"死亡时刻"记录的不是同一时间——暗示记忆的时间轴被篡改过。

**关键发现**：
- 记忆化身的行为不一致 → 梦境数据被"拼接"过
- 台词偏差 → 存在两个不同"版本"的陆震霆记忆
- 时钟不一致 → 时间轴存在断裂或重叠
- 这些异常共同指向"记忆被篡改"的结论

**线索获取**：
- 公开线索：时钟不一致（可被所有人观察到）
- 专属线索：
  - 林若雪：感知到版本1台词的情感（疲惫、自我怀疑）与版本2台词的情感（决心、接受命运）完全不同——它们是同一个人在不同精神状态下的表现
  - 周明远：判断服装切换的规律——切换间隔非常精确（约3分17秒），这是"数据拼接"的技术特征（不同源文件的时间长度差异导致）
  - 陈墨：注意到记忆化身的行走路径——每次切换后，路径会有微小偏移，仿佛"走错了位置再纠正"

**后续影响**：
- 确认记忆存在异常 → 推动调查员寻找原因
- 为E-03（记忆黑洞发现）做铺垫——异常现象在同一个区域（书房）集中出现，引导调查员深入搜索书房

**其他触发方式**：
如果调查员没有主动进入书房观察，约30分钟后，一个自动触发会生效——赵天宇会感受到一阵"眩晕"，指向书架后方的异常区域，引导团队前往。

### E-03：第一个记忆黑洞与闪回【主线·关键】

**事件编号**：E-03
**所属幕次**：第一幕（约30-40分钟时触发）
**触发条件**：调查员探索书房的"书架后方"区域（移开书架上的书或绕到书架侧面）
**类型**：探索触发

**事件描述**：

**第一步——发现异常空间**：
在书房东墙的书架与墙壁之间（需要移开几本书——这些书在触碰时"感觉像棉花"，因为陆震霆很少翻动它们），存在一个约1.5米高、0.8米宽的完全黑暗区域。这片黑暗"不自然"——它不是"没有光"的黑暗，而是"什么都没有"的绝对虚空。

**第二步——感知异常**：
- 靠近黑暗边缘（距离约2米）时：所有调查员感受到一种难以言喻的"不适感"——仿佛站在悬崖边缘，或者靠近一个巨大的空洞
- 更靠近（距离约1米）：皮肤感觉到轻微的刺痛感（针扎感），空气中弥漫着"烧焦"的味道——但无法确定是嗅觉还是大脑直接产生的感知
- 接触到黑暗边缘（距离0米）：强烈的情绪冲击——混杂着恐惧、愤怒、背叛感和绝望的"情绪风暴"直接涌入调查员的意识

**第三步——闪回触发**：
当第一次有调查员的手触碰到黑暗边缘（或被情绪风暴推入边缘）时，触发"集体闪回"：
1. 所有人的视野瞬间切换——书房消失，取而代之的是一段模糊的画面
2. 画面内容：一个布满仪器的地下实验室，全息屏幕发出蓝绿色的光，屏幕上是密密麻麻的代码和一个人影的轮廓
3. 声音（失真、扭曲、仿佛在水下听到的）："……越过了底线……我们……不能这样做……"
4. 画面剧烈抖动，然后"崩毁"——像玻璃一样碎裂
5. 调查员的视野回到书房，所有人都感到剧烈的头痛和恶心，有些人（赵天宇最严重）会短暂失去平衡

**第四步——闪回后的痕迹**：
在闪回结束后，黑暗区域边缘出现了一些"画面残影"——刚才闪回中的实验室画面碎片像烧焦的胶片一样漂浮在黑暗边缘，持续约3秒后消散。

**关键发现**：
- 记忆黑洞 → 梦境数据被人为切除（周明远可以确认这是"神经切割"的典型痕迹）
- 闪回内容 → 实验室场景（"DMB-L7"编号在画面中一闪而过）
- 情绪残留 → 陆震霆在被切除记忆区域残留的深层情感（恐惧+愤怒+绝望）
- 烧焦的气味和边缘的锯齿状 → 高能神经切割工具留下的痕迹

**线索获取**：
- 核心线索：记忆被"切除"的确凿证据
- 线索卡片：闪回中捕捉到的细节——"全息屏幕上的实验室编号"DMB-L7""
- 专属线索：
  - 林若雪：感受到情绪风暴中有一个"名字"——是"沈牧之"还是"沈"——她不能完全确定，但能感知到某种与"沈"相关的强烈情绪（愤怒和背叛感）
  - 周明远：可以判定记忆黑洞的切除手法——"高精度神经束切割，不是粗放的删除，是精细的外科手术级别——这是DEAS最高管理员权限才有的操作"
  - 苏雨桐：监测到记忆黑洞区域散发的"精神污染指数"远超正常值——建议远离，并提醒团队该区域对意识有侵蚀作用

**后续影响**：
- 触发E-04（无面守护者出现）——几乎必然的连锁反应
- 触发E-05的线索链条——闪回中的"DMB-L7"编号指向隐藏入口的位置
- 为第二幕的DMB-L7实验室场景提供"入口指引"

### E-04：无面守护者首次袭击【主线·威胁】

**事件编号**：E-04
**所属幕次**：第一幕（紧接E-03触发，约2分钟后）
**触发条件**：E-03中第一次触碰记忆黑洞边缘
**类型**：自动触发

**事件描述**：

**阶段一——空间震荡**：
在闪回结束的瞬间，整个书房开始剧烈震动。书桌上的物品（笔、纸张、那台老式打字机）开始跳动。书架的书籍开始掉落——但掉落的书在落地前就"消失"在空中——因为它们在梦境中不是"实体"。

**阶段二——墙壁剥落**：
书房正对着记忆黑洞的那面墙壁开始异常。墙纸像纸片一样一片片剥落——但剥落的不是"墙纸"，而是"墙壁本身"。每一片剥落后的区域露出的不是砖石，而是纯粹的黑色——仿佛墙壁后方是另一个维度。

**阶段三——守护者降临**：
从墙壁的缺口处，一个纯黑色的人形轮廓缓缓"流"出来。它没有五官、没有衣物纹理、没有头发——只有一个完美的人形剪影，表面像液态的黑色金属。它的移动方式不符合物理规律——仿佛是在地面上方几毫米处"滑动"。

**阶段四——对峙与消失**：
1. 守护者向调查员的方向移动，速度不快但无法阻挡
2. 苏雨桐激活锚定物（精神屏障装置），展开一道蓝色半透明屏障
3. 守护者在屏障前停住——不是被挡住的，而是它自己选择了停下
4. 它"转动"头部，在在场所有人中"扫描"——虽然没有眼睛，但可以明确感知到它在逐一"注视"每个人
5. 它在赵天宇身上"停留"了约3秒——这期间赵天宇感到剧烈的头痛，仿佛有什么东西在他的脑子里"搅动"
6. 然后守护者缓缓后退，融入墙壁的黑暗中
7. 墙壁恢复——墙纸"长回"原位——一切恢复正常

**关键发现**：
- 守护者存在 → 梦境中有主动防御机制
- 守护者的行为模式 → 它会阻止调查员接触记忆黑洞
- 守护者对赵天宇的特殊关注 → 赵天宇与守护者存在某种"联系"
- 苏雨桐的屏障有效 → 精神防御可以抵御守护者（但能承受多久？）

**线索获取**：
- 公开线索：守护者出现/消失的全过程
- 专属线索：
  - 赵天宇（最重要）：在守护者"注视"他时，他除了头痛，还"听到"了一个微弱的、只有他能听到的声音，声音说了一句话（内容根据赵天宇的选择可以不同——默认是一个词："停下"——但这是一个线索：声音是沈牧之的，赵天宇在入职培训时听过他的讲座，应该能辨认出来）
  - 苏雨桐：她的屏障在对抗守护者时，监测到了守护者的"能量构成"——这不是梦境自然生成的防御机制，而是有"外部意识注入"的痕迹（这是关键线索——守护者被人为操控）
  - 周明远：如果他在守护者出现时携带了便携式神经信号扫描仪，他可以捕捉到一组"外源神经信号"——这与梦境本地信号不同，像是从外部"入侵"的

**后续影响**：
- 确认守护者的威胁性 → 团队需要更小心地探索
- 赵天宇的特殊性开始浮现 → 可能成为团队中的"不确定因素"
- 隐藏影响：沈牧之在现实中确认了调查员的探索深度和方向

### E-05：发现隐藏入口【主线·过渡】

**事件编号**：E-05
**所属幕次**：第一幕（约40-60分钟时触发）
**触发条件**：调查员综合闪回线索，在地下酒窖找到隐藏门
**类型**：推理触发 / 探索触发

**事件描述**：

**线索整合过程**：
调查员需要将以下线索串联起来：
1. 闪回画面中的"DMB-L7"字样 → 这应该是一个实验室的编号
2. 观澜山庄的建筑图（存在于书房抽屉中的原始设计图）→ 显示别墅地下室的面积比实际看到的"少了约2.7%"
3. 在酒窖中的深入观察 → 北墙的湿度与其他墙壁不同，敲击声有空洞回音
4. 移开酒柜后 → 发现一扇与墙壁几乎融为一体的金属门

**门的解锁方式**：
在金属门上，有一个全息扫描器。调查员需要解决"如何通过陆震霆的生物识别认证"：
- 方式A（简单）：周明远可以用技术手段模拟陆震霆的神经特征——需进行一次技术判定，成功率为60%（如果有林若雪的共情感知辅助，提高到80%）
- 方式B（推荐——更有故事性）：林若雪尝试"共情"陆震霆开门时的记忆——触摸门的过程中，她感知到了陆震霆站在这里时的情感（紧张、决心、恐惧），通过这些情感"引导"门识别了陆震霆的意识身份

**过渡效果**：
门开启的瞬间，整个梦境场景开始"融化"：
1. 调查员脚下的酒窖地砖变成金属网格地板
2. 空气中的葡萄酒香化为消毒水和电子设备的气味
3. 头顶的木质横梁消失，露出布满管线的金属天花板
4. 墙壁上的装饰剥落，转为银灰色的金属
5. 身后——酒窖的门消失了——他们再也回不到观澜山庄了（或者说，他们已经不在第一层了）
6. 前方——一条约20米长的金属走廊，尽头是"DMB秘密研究分部 — L7实验室"的大门

**关键发现**：
- 观澜山庄地下存在一个隐藏的DMB实验室
- 进入这里意味着离开了表层记忆层，进入了深层梦境层
- 身后的路消失了——探索变成了"有进无退"

**线索获取**：
- 建筑图与实地对比 → 证明隐藏空间的存在
- 门的全息屏显示 → 最后一次开启记录是"2089年7月14日 21:04"（案发前约1小时）——陆震霆在死前还进入过实验室
- 门禁系统的操作日志 → 显示陆震霆的权限等级为"最高级（L10）"——高于沈牧之的"管理级（L9）"

**后续影响**：
- 触发第二幕的开启 → 场景转入DMB-L7实验室
- 调查员的不归路 → 无法返回第一层，只能向前探索
- 为E-07（实验室探索）做好准备

### E-06：进入深层梦境层【主线·过渡】

**事件编号**：E-06
**所属幕次**：第一幕结束/第二幕开始
**触发条件**：通过隐藏入口进入实验室走廊
**类型**：自动过渡

**事件描述**：
这是幕次之间的过渡事件，包含两个功能：

### R-01：记忆被篡改的确认

**反转编号**：R-01
**触发时机**：第一幕中期（E-02或E-03后）
**前置条件**：调查员观察到记忆化身的不一致性，或发现第一个记忆黑洞

从"陆震霆的死亡可能不简单"到"陆震霆的记忆被人为篡改"的认知转变。

**反转前的信息状态**：
- 调查员知道：陆震霆死了、官方结论是意外、梦境数据被标记为异常

**反转后的新目标**：

- 找到"为什么这么做？" → 需要找到隐藏的线索（指向实验室）

**关键触发元素**：
- E-03中的记忆黑洞闪回——直接证据
- E-04中的守护者出现——间接证据
- BE-01中的日记——辅助证据

**冲击力设计**：
- 第一次闪回的视觉冲击（突然的实验室场景切换）配合情绪冲击（情感风暴中的恐惧和愤怒）
- 玩家从"我们在探索一个案发现场"突然切换到"我们在探索一个有意识的世界"
- 守护者的出现提供了"危险正在逼近"的感觉——不只是解密，还有生存压力

**反转前的信息状态**：

- 理解"终极真相"——这个案子的背后不是一个人类阴谋家，而是一个数字意识的自主行动
- 面对"数字陆震霆"——是敌？是友？还是可以利用的资源？
- 做出终极选择——与"数字陆震霆"合作？还是将其与沈牧之一同锁入梦境？

**关键触发元素**：
- BE-05中数据核心的意识脉冲——直接证据
- BE-06中加密通讯日志中的"LZ"身份分析——间接证据
- E-09中争吵记忆里沈牧之说的"在我之上"力量——回溯时重新理解
- 陆震霆的全息留言中提到的"系统里的东西"——线索串联

**冲击力设计**：
- 这是剧本中最深层的反转——颠覆所有之前的认知
- "数字陆震霆"的出现将道德问题复杂化——他到底算不算"陆震霆"？他有没有生存权？
- 玩家需要重新评估整场事件——"陆震霆真的是彻底的受害者吗？" "沈牧之真的是彻底的恶人吗？"
- 开放的哲学问题："数字意识"享有与人类相同的权利吗？
- 这个反转不一定会被所有玩家触发——设置为"隐藏的终极秘密"——增加了重玩价值
