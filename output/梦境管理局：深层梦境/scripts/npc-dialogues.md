## 1. 陆震霆记忆化身 — 循环台词与偏差系统

### 1.1 基本信息

| 属性 | 内容 |
|------|------|
| **NPC名称** | 陆震霆记忆化身 |
| **本质** | 死者记忆数据的自动回放投影，非自主意识 |
| **位置** | 表层记忆层，观澜山庄书房（主要活动范围） |
| **出现时段** | 第一幕全程（约60-70分钟游戏时间） |
| **外观** | 半透明发光轮廓，身形清晰但面部表情在切换时有短暂重影。脚步声与身体移动有约0.3秒延迟。不会对调查员的出现做出任何反应。 |
| **DM提示** | 说话时语速应带有"梦呓"般的飘忽感。每句台词之间保持约2秒的呼吸间隔，营造记忆回放的循环质感。偏差台词（Line 5和Line 8）切换时，语气和体态应出现可被观察的断崖式变化。 |

### 1.2 循环行为背景

记忆化身执行三套循环行为（书桌前阅读 / 书架前踱步 / 壁炉前沉思），每套约3分17秒。台词在不同循环中穿插出现。正常台词（1/2/3/4/6/7）在梦境中自然回放；偏差台词（5/8）是记忆篡改后留下的"时间裂缝"——不同版本的回放片段被错误地拼接在一起。

### 1.3 台词序列与表演指导

DM按照以下8句台词循环播放。每句台词后标注了调查员可能观察到的视觉细节和偏差特征。

---

#### 【正常循环台词 — 梦境稳定态】

**Line 1** — 书桌前阅读状态

| 项目 | 内容 |
|------|------|
| **台词** | "这个算法……还需要再优化一下……" |
| **出现时机** | 记忆化身坐在书桌前，翻开一本神经科学期刊，目光扫过页面时低声自语 |
| **语气** | 专注、喃喃自语、略带疲惫 |
| **体态** | 右手握笔，笔尖悬停在页边空白处。目光从文字上抬起，望向窗外，仿佛在脑中推演 |
| **视觉细节** | 书籍标题模糊不可辨读。台灯光线在桌面上投下一个温暖的圆形光斑 |
| **调查员可观察到的异常** | 笔尖划过纸面的声音循环重复——仔细听会发现声音波形完全一致，这是记忆数据压缩的特征 |
| **DM表演指导** | 语速偏慢，声音中透出沉思想象的飘忽感。"算法"二字略微重读，"再优化一下"逐渐变轻，仿佛注意力已经移开 |

---

**Line 2** — 走向书架的过渡状态

| 项目 | 内容 |
|------|------|
| **台词** | "我记得……我把它放在哪里了？" |
| **出现时机** | 记忆化身从书桌起身，走向东墙书架。在书架前站定，目光扫过书脊时说出 |
| **语气** | 困惑、轻微的自嘲、自言自语 |
| **体态** | 右手抬起，食指在一排书脊上缓缓滑过。头微微歪向左侧，眉头微蹙 |
| **视觉细节** | 书架上的书籍编号大多模糊，只有几本陆震霆经常翻阅的书有相对清晰的轮廓 |
| **调查员可观察到的异常** | 食指滑过的轨迹出现0.5秒的"跳帧"——手指突然出现在前方3厘米处，中间动作缺失 |
| **DM表演指导** | "我记得"之后做一个自然的停顿（约1.5秒），仿佛在搜索记忆。整句台词带上一点老年人般的缓慢，尽管陆震霆只有58岁——这是疲惫的表现 |

---

**Line 3** — 壁炉前沉思状态

| 项目 | 内容 |
|------|------|
| **台词** | "时间不多了……" |
| **出现时机** | 记忆化身站在壁炉前，右手扶着壁炉台，凝视跳动的火焰时低声说出 |
| **语气** | 低沉、沉重、带有一种隐晦的紧迫感 |
| **体态** | 静止。整个人像一尊雕塑，只有火光在脸上投下跳动的阴影。右手无意识地握紧壁炉台边缘 |
| **视觉细节** | 壁炉中的火焰是整栋别墅中唯一"有温度"的区域——它稳定地燃烧，没有烟，没有灰烬 |
| **调查员可观察到的异常** | 如果林若雪使用情感共鸣，会感知到这句台词附着的情感与陆震霆当前的表面情绪（焦虑）不同——底层有一种"接受"和"释然" |
| **DM表演指导** | 声音压得很低，像是说给自己听的。不要看任何人（即使有调查员站在面前），目光始终锁定在火焰上。"时间"二字带一点咬牙的力道，"不多了"则几乎是叹息 |

---

**Line 4** — 书房踱步状态

| 项目 | 内容 |
|------|------|
| **台词** | "我需要想清楚……" |
| **出现时机** | 记忆化身从书架走向窗户，在窗前停住，望着窗外的海景时说出 |
| **语气** | 犹豫、自我怀疑、内心斗争的语调 |
| **体态** | 双手插在睡衣口袋里，肩膀微塌。背对房间，面朝窗户——窗外是黑色的海面 |
| **视觉细节** | 窗外的海在梦境中是静止的——没有波浪，没有月光在水面上的反射。天空呈现不自然的淡紫色 |
| **调查员可观察到的异常** | 这句台词与Line 5的内容形成矛盾——"我需要想清楚"（犹豫）vs "我必须做个了断"（决绝），两种心态在记忆中被错误地拼接在一起 |
| **DM表演指导** | 这句台词需要一个深呼吸作为前奏（现实中的陆震霆在说这句话前吸了一口气）。声音中带上一种努力说服自己的微颤 |

---

**Line 6** — 壁炉前回忆状态

| 项目 | 内容 |
|------|------|
| **台词** | "当年我们创立深梦科技的时候……不是这样的……" |
| **出现时机** | 记忆化身站在壁炉前，从口袋中掏出一枚干银杏叶，夹在手指间转动时说出 |
| **语气** | 怀念、悲伤、遗憾交织的复杂情绪 |
| **体态** | 低头看着手中的银杏叶。他的拇指轻轻摩挲着叶片的边缘——一个充满情感慰藉的重复动作 |
| **视觉细节** | 银杏叶在火光中呈现金黄色。叶片保存得很完整，显然被精心处理过 |
| **调查员可观察到的异常** | "不是这样的"这句话的情感强度与他周围的记忆层数据密度不符——这段记忆的情感太"浓"了，与其他"清淡"的记忆形成对比。这说明这段记忆对陆震霆极为重要 |
| **DM表演指导** | 在"深梦科技"四字上带上微弱的笑意（记忆中的温暖刹那），然后在"不是这样的"处让声音低沉下去。中间的停顿（……）保持4-5秒，让情感在沉默中发酵 |

---

**Line 7** — 窗边眺望状态

| 项目 | 内容 |
|------|------|
| **台词** | "海的那边……是什么来着？" |
| **出现时机** | 记忆化身站在书房的落地窗前，右手按在玻璃上，目光望向远方的海平线 |
| **语气** | 恍惚、失神、仿佛在寻找一个被遗忘的事物 |
| **体态** | 整个人几乎静止，只有按在玻璃上的手指微微张开又合拢——一个无意识的抓握动作 |
| **视觉细节** | 他的呼吸在玻璃上形成了一层薄雾——但雾气在梦境中不会消散，而是保持定格。透过雾气，窗外的海平线处有一丝微光 |
| **调查员可观察到的异常** | 这句台词中"那边"二字的音调略微上扬——这不像是在问自己，更像是在问某个"想象中的倾听者"。这是"数字陆震霆"在记忆中留下的微弱痕迹——本体在无意识中感知到系统中有另一个"自己" |
| **DM表演指导** | 这句台词需要最轻的音量——几乎是耳语。"海的那边"带有童真的好奇（联想到他5岁时在海边的记忆），"是什么来着"尾音飘散，像是答案已经消失在风中 |

#### 【偏差台词 — 记忆篡改裂缝】

> **重要提示**：Line 5和Line 8是"偏差版本"，它们在正常循环中约每3次循环出现1次。它们的出现频率会随着调查员在书房中的探索深度而增加——调查员越接近记忆黑洞，偏差台词出现得越频繁。当调查员首次触发记忆黑洞（E-03）后，所有后续循环中偏差台词的出现概率提升至50%。

---

**Line 5** — 偏差版本1：睡衣版的决绝

| 项目 | 内容 |
|------|------|
| **台词** | "我必须做个了断。" |
| **出现时机** | 取代Line 4的位置出现。记忆化身同样站在窗前，但体态完全不同 |
| **语气** | **断崖式转变**——从犹豫变为决绝、坚定、甚至带着一丝冷酷。与Line 4的"我需要想清楚"形成尖锐对比 |
| **体态** | 不再塌肩——脊背挺直，下巴微抬。双手从口袋中抽出，自然垂放身侧——一个准备行动的姿势。眼神不再是迷茫的——是聚焦的、坚定的 |
| **服装状态** | 仍然是灰色丝绸睡衣——与循环A和循环C的服装一致。但偏差在于动作——穿睡衣的陆震霆不应该有这种"决断"的姿态。这引起了视觉与情绪的不协调 |
| **调查员可观察到的异常** | 如果调查员同时观察到了Line 4和Line 5，可以发现两句台词在记忆化身身上"打架"——有时前一句还是"我需要想清楚"，后一句就切换为"我必须做个了断"，像两个人的对话被错误地塞进了同一个人的嘴里 |
| **DM表演指导** | 这是全剧第一次"记忆裂缝"的明确信号。表演时需要让玩家感受到明显的"卡顿"——Line 5出现前，记忆化身会有一个约1秒的"定格"（仿佛系统在切换数据源），然后语气、体态、眼神全部同时切换。让玩家有一种"同一张脸，完全不同的灵魂"的毛骨悚然感 |

---

**Line 8** — 偏差版本2：西装版的忏悔

| 项目 | 内容 |
|------|------|
| **台词** | "越过了底线……我们……不能这样做……" |
| **出现时机** | 取代Line 6的位置出现。记忆化身同样在壁炉前拿着银杏叶，但场景出现短暂的不稳定 |
| **语气** | **严重失真**——声音扭曲、带有电子失真感（像从很远的地方传来的、被压缩过的音频）。情绪上充满痛苦、恐惧、忏悔 |
| **体态** | 身体微微颤抖。拿着银杏叶的手在发抖。他的视线没有固定在任何物体上——而是"穿过"了壁炉、墙壁，望向某个不存在于此地的场景 |
| **服装状态** | **蓝色商务西装**——与偏差发生时的环境（壁炉前、拿银杏叶、穿睡衣的正常版本）完全不符。这说明这段记忆碎片来自另一个时间线（陆震霆白天在办公室时的记忆），被错误地拼接到了夜晚的场景中 |
| **视觉细节** | 这句台词出现时，整个书房的色调会短暂闪烁——从暖黄色变为冷蓝色（约0.5秒），然后恢复。壁炉中的火焰会短暂变成蓝色 |
| **调查员可观察到的异常** | 这句台词是"闪回"的前奏——它对应的是被切除的记忆A中的内容（陆震霆在DMB-L7实验室中对沈牧之说的话）。如果调查员在此后探索书房的书架后方，记忆黑洞的发现会更加"顺理成章" |
| **DM表演指导** | 这是全剧最重要的"记忆裂缝"信号。表演时需要制造出强烈的"异质感"：声音可以采用略微压低+改变共鸣位置的方式，让声线听起来比正常台词"远"一些。"越过了底线"四个字咬牙着力，"我们"之后有一个长长的、充满颤抖的停顿（3-4秒），最后的"不能这样做"几乎变成气声——仿佛说话的人正在经历巨大的精神痛苦。说完后，记忆化身会有一个约2秒的"恢复期"——身体晃了晃，然后继续以正常状态执行下一句循环 |

### 1.4 偏差触发机制速查表

| 触发条件 | 效果 | 备注 |
|---------|------|------|
| 调查员进入书房 | 正常循环开始 | 循环A→B→C→A... |
| 调查员观察记忆化身超过3分钟 | 首次出现台词偏差（Line 5或Line 8随机） | 偏差后恢复正常循环 |
| 调查员触碰记忆黑洞边缘（E-03） | 偏差出现频率提升至50% | 每次循环都有50%概率出现偏差台词 |
| 调查员深入探索记忆黑洞区域 | 偏差出现频率提升至100% | 每次循环必然包含至少一条偏差台词 |
| 调查员离开书房区域 | 偏差频率重置 | 再次进入时重新从正常循环开始 |

## 2. 沈牧之降临对话 — 完整对话树

### 2.1 基本信息

| 属性 | 内容 |
|------|------|
| **NPC名称** | 沈牧之（意识投影） |
| **本质** | 通过DEAS系统投射到梦境中的活体意识，具有完整自主性 |
| **出现时段** | 第二幕末至第三幕（E-11触发后），约在游戏进行到130-170分钟时 |
| **出现位置** | DMB-L7实验室主控室 |
| **形态变化** | 初始为黑色人形（无面守护者升级版），逐渐细化出完整的沈牧之外貌 |
| **DM核心提示** | 沈牧之的智力傲慢是其最大特征。他不屑于说谎——他用选择性真相和语言框架来操控调查员的认知。在表演时保持冷静、理性、略带学术演讲者的从容，但在被触及"数字陆震霆"相关话题时应出现微妙的紧张。 |

### 2.2 降临流程

```
【阶段一】前兆（约30秒）
实验室灯光闪烁 → 全息屏幕雪花干扰 → 温度骤降5°C → 空气中出现臭氧味

【阶段二】汇聚（约30秒）
地面接缝处渗出黑影 → 黑影汇聚成人形轮廓 → 轮廓表面出现纹理细节

【阶段三】显形（约15秒）
白大褂、面部特征、眼镜被"绘制"完成 → 投影睁开眼 → 第一次"看向"调查员

【阶段四】对话（约5-8分钟，DM根据实际节奏控制）
沈牧之开场 → 调查员应对 → 分支对话 → 退场

【阶段五】退场（约15秒）
投影逐渐变淡 → 倒计时数字首次浮现 → 声音留下最后一句话后消散
```

### 2.3 完整对话树

---

#### A. 沈牧之开场（自动触发，不可跳过）

> **沈牧之**：（投影稳定后，环视在场的每一个调查员，目光在每个人脸上停留约1秒。他的声音平静、清晰，带着一种"学术报告"式的从容）
>
> "晚上好——或者说，梦境里的时间概念没什么意义。我是沈牧之。DMB首席科学家，DEAS系统的发明者，以及——你们已经猜到的——陆震霆的……终结者。"
>
> （停顿——他观察调查员的反应，嘴角浮现一丝微不可察的笑意）
>
> "我猜你们已经看了保险箱里的东西。织梦者计划、交易记录、DMB高层的会议录音……嗯，陆震霆准备得很周全。他是一个细致的人——和我共事二十年，我比任何人都了解这一点。"
>
> "但我现在站在这里，不是为了辩解。我只是想给你们——一个选择。在你们做出决定之前，让我先说说……我的版本。故事的另一个版本。"

---

#### B. 他的版本——织梦者计划

> **当调查员没有打断时，沈牧之继续：**
>
> "你们看到的档案是怎么描述'织梦者'的？'非法人体试验'、'意识操控武器'、'洗脑工具'——这些词我都见过。你们想知道真相吗？"
>

### 6.2 对话节奏控制要点

1. **沈牧之的对话不要超过8分钟**：时间过长会让玩家失去张力。如果调查员提出很多问题，选择最重要的3-4个让沈牧之详细回答，其余用"这个问题你们之后会找到答案的"带过。

2. **林晓的对话建议分两次进行**：第一次获取基本信息（话题1-3），第二次（如果调查员再次探访隔间）再深入核心线索（话题5-7）。这样信息密度适中，玩家更容易吸收。

3. **"数字陆震霆"的对话是全员广播**：当调查员进入数据裂缝并与它建立联系时，所有在梦境中的调查员都能"听到"它的声音（直接意识传输）。所以不需要让某个调查员转述。

4. **偏差台词的渐进式频率**：第一次进入书房时，偏差台词（Line 5/8）出现概率约30%。触发记忆黑洞后提升至50%。之后每次回到书房，偏差频率递增10%，直到100%——模拟梦境因深层探索而变得越来越不稳定。

5. **管家最多交互3次**：第3次交互后NPC卡死并消散——告诉调查员"他已经被过度询问导致投影崩溃了"，并提醒玩家记忆投影类NPC有交互上限。

### 6.3 紧急情况处理

| 紧急情况 | 处理方案 |
|---------|---------|
| 调查员试图攻击沈牧之的投影 | 攻击穿过投影——沈牧之平静地说："这是在浪费你宝贵的时间。"然后继续对话。如果攻击持续，沈牧之提前结束对话并激活倒计时 |
