- **灯光/视觉**: 以 💡 标记，提供线下灯光建议和线上视觉辅助方案
- **BGM推荐**: 以 🎵 标记，提供风格描述和曲目推荐（不指定具体版权曲目，提供搜索关键词）

# 第一幕：欢迎来到观澜山庄

**整体时长**: 约60-90分钟
**氛围等级**: ★★★★☆（神秘 + 不安 + 好奇）

## 第一幕整体氛围设计

| 维度 | 设计说明 |
|------|---------|
| **视觉主色调** | 暖琥珀色 + 淡紫色阴影（梦境特有的色彩偏移） |
| **温度感** | 体感"偏暖但有不自然的凉意"——老宅中的阴翳 |
| **气味联想** | 老书、木质家具、壁炉的烟火气、以及——某种微弱的电子设备的气息（与背景年代吻合） |
| **声音基底** | 低频嗡鸣（DEAS系统运行的背景音）+ 远处隐约的海浪声 + 偶尔的木质家具"咯吱"声 |

## 第一幕关键场景氛围分解

### 场景一：简报室集结

**氛围要求**: 专业、冷静、略带压力的任务开端

- **🎭 陈墨简报**: 声音需要有一种"录制好的平静"——不是没有感情——是把感情压在了水面下
- **🎭 赵天宇的紧张提问**: 他声音中应该有一丝不易察觉的颤抖——给玩家一个"这个新人还不成熟"的印象
- **🎭 方晴的介入**: 她的声音来自通讯器——可以用轻微的电子音效处理——强化"外部支援"的距离感

**音效设计**:
- ⚡ 开场: 低频嗡鸣音 (40Hz 正弦波，极低音量) ——持续背景
- ⚡ DEAS舱体闭合: 气动密封声 + 机械锁扣声
- ⚡ 神经接口连接: 一组快速上升的电子音调——然后沉默
- ⚡ 意识投射瞬间: 心跳声加速——然后一声深长的"坠落"音效——然后安静

---

### 场景二：第一次发现异常

**氛围要求**: 从平静到困惑——"有什么东西不对"

- **🎭 赵天宇发现服装切换**: 他的声音应该从随意的观察变为困惑——"等等——"
- **🎭 周明远的技术分析**: 他的兴奋感应该让玩家感到"这个技术狂人发现了什么了不起的东西"——而不是恐慌
- **🎭 林若雪的情感解读**: 她的语气应该带着一种"真相让人悲伤"的质感

**音效设计**:
- ⚡ 观察循环: 模拟怀表的滴答声（极轻）——每3分17秒出现一次短暂的"时间跳跃"音效（一种轻微的"嗖"声——暗示记忆拼接的缝隙）
- ⚡ 台词偏差发现: 当林若雪提到"两种不同的语气"时——加入短暂的双声道效果——左右耳切换到不同语气
- ⚡ 书架后方黑暗: 当赵天宇说"这边有点不对劲"时——混响效果缓慢增加——暗示空间的异常

**灯光提示 💡**:
- 正常观察时: 暖光
- 第一次发现异常: 灯光轻微闪烁一次
- 靠近书架后方暗区时: 灯光缓慢变暗——但只限于舞台的"书架区域"

---

### 场景三：记忆黑洞闪回

**氛围要求**: 突然的冲击——感官过载——然后恢复

- **🎭 靠近黑暗**: 每个调查员的反应应该分层——陈墨的警觉、赵天宇的好奇、林若雪的谨慎、周明远的科学兴趣
- **🎭 闪回触发**: 全息画面的突然切换需要给玩家带来真实的"跳吓"效果
- **🎭 闪回后的反应**: 混乱和不适——玩家应该感到"我们的意识被强行拖入了别人的记忆中"

**音效设计**:
- ⚡ 靠近黑暗: 低频嗡鸣逐渐增强——伴随微弱的"烧灼"声效
- ⚡ 触碰瞬间: 一声尖锐的耳鸣音——然后全频段白噪音——然后画面切入
- ⚡ 闪回中: 声音扭曲效果（像是从水下听到的）+ 电子失真——画面中的"底线"一词被拉长
- ⚡ 闪回结束: 玻璃碎裂的音效——然后一切突然安静——只剩下沉重的呼吸声

**灯光提示 💡**:
- 靠近记忆黑洞: 聚光——只有调查员的区域还有光——周围渐暗
- 闪回触发: 全场灯光熄灭——然后闪烁的蓝白色光——节奏越来越快——然后突然全黑——恢复

---

### 场景四：守护者首次出现

**氛围要求**: 压迫感 + 未知的恐惧

- **🎭 空间震荡**: 书桌上的物品跳动——声音上的混乱
- **🎭 守护者从墙壁中"渗出"**: 这是一个视觉和听觉上的"不可能"事件——强调它的非人感
- **🎭 苏雨桐激活屏障**: 蓝色光芒的效果——在黑暗中是唯一的安全象征
- **🎭 守护者"注视"赵天宇**: 这是全场最紧张的时刻——赵天宇的反应要真实到让玩家们屏住呼吸

**音效设计**:
- ⚡ 空间震荡: 低频轰鸣 + 玻璃震颤声
- ⚡ 墙壁剥落: 纸张撕裂声——但被放大和扭曲——变得不自然
- ⚡ 守护者从墙壁渗出: 粘稠液体流动的声音——但没有任何水分——更像"数据流"的声音效果
- ⚡ 守护者移动: 无脚步声——但在它经过的位置——有一丝静电干扰声
- ⚡ 守护者"注视"赵天宇: 赵天宇耳机中出现的低频耳语——其他玩家听不到——只有DM通过耳机（在物理跑团中，DM可以走到赵天宇玩家身后低语）
- ⚡ 守护者消失: 一声低沉的"叹息"——墙壁恢复的"长回"声——然后用沉默确认它的离开

**灯光提示 💡**:
- 震荡阶段: 灯光闪烁——混乱的光影
- 守护者出现时: 所有灯光突然变暗——只剩紧急备用光的冷白色——在守护者身上投下长影
- 苏雨桐激活屏障: 灯光中混入蓝色调——在屏障范围内形成一个"安全岛"
- 守护者消失: 灯光缓慢恢复正常——但留下一丝"不可见的寒意"

---

### 场景五：发现隐藏入口

**氛围要求**: 探索的兴奋 + 决策的重量

- **🎭 发现门**: 应该有"终于找到了"的成就感
- **🎭 林若雪开锁**: 这是第一次——玩家看到林若雪的"特殊能力"在团队中发挥作用
- **🎭 陈墨的"选择"**: 这是剧本的第一个正式决策点——陈墨的语言需要让玩家感到"这是一个有重量的选择"

**音效设计**:
- ⚡ 全息扫描器激活: 清脆的电子音 + 扫描光效的声音
- ⚡ 门打开: 沉重的金属门闩声——压缩空气释放声——然后金属摩擦声
- ⚡ 门后的声音: 从门缝中传出——一种低沉的、持续的嗡鸣——跟别墅的安静形成鲜明对比——是实验室设备的声音

**灯光提示 💡**:
- 酒窖: 昏暗的暖色——偶尔闪烁的灯光（像是老旧的灯泡）
- 门打开: 从门缝中泄出冷白色的荧光灯光——在酒窖的暖色中形成一道白色的光剑
- 进入走廊: 灯光从暖色完全切换为冷白荧光灯——标志着"你已经离开了"

## 第二幕关键场景氛围分解

### 场景六：争吵记忆回放

**氛围要求**: 两个人的悲剧——而不是纯粹的正邪对立

- **🎭 全息影像中陆震霆的愤怒**: 他的愤怒不是吼叫——是压抑的、沉重的一字一句
- **🎭 沈牧之的回避**: 当他说"在我之上"时——玩家应该感到"他说的不是DMB高层——是别的什么"
- **🎭 回放结束后的沉默**: 这应该是第二幕中第一个"所有人都不说话"的时刻

**音效设计**:
- ⚡ 相框触碰: 一种温柔的"激活"音效——像是老式收音机调频的声音
- ⚡ 全息回放中: 轻微的房间混响——让记忆中的对话听起来"从远处传来"
- ⚡ 陆震霆说"你忘了吗": 他的声音在这一刻——混响突然消失——变得清晰而锐利——强调这句话的情感冲击
- ⚡ 回放结束: 一声轻柔的"咔哒"声——相框回到静止状态——然后是沉重的沉默

**灯光提示 💡**:
- 全息回放时: 房间灯光调暗——全息投影的蓝白色光成为唯一光源——在玩家脸上投下跳动的光影
- 关键台词时刻: 在陆震霆说"你越过底线了"时——灯光短暂闪烁——增加冲击力
- 沈牧之说"在我之上"时: 灯光突然变冷——一瞬的蓝色调——暗示这句话背后隐藏的寒意

---

### 场景七：林若雪发现胸针秘密

**氛围要求**: 个人层面上的真相揭露——温暖与愧疚交织

- **🎭 林若雪的坦白**: 她的声音应该有"我一直想说出这个秘密——但不知道如何开口"的质感
- **🎭 苏雨桐的质问**: 不应该是攻击性的——应该是"我必须确认"的专业态度
- **🎭 胸针展开**: 这是一个"礼物中的礼物"——视觉上应该是美的——即使它承载着沉重的真相

**音效设计**:
- ⚡ 胸针开始发光: 极其轻柔的"水晶"音效——高音调、透明
- ⚡ 胸针展开: 精密机械运动的声音——齿轮咔嗒——锁定——然后展开
- ⚡ 芯片露出: 一声轻微的"嗡"——纳米级设备激活的声音

**灯光提示 💡**:
- 胸针发光: 从林若雪胸前——一点蓝光开始扩散——在她的脸上投下蓝色的光晕
- 其他区域渐暗——观众的注意力被引导到胸针和保险箱门上

---

### 场景八：数据芯片激活——陆震霆的全息留言

**氛围要求**: 跨越生死的对话——死者最后的嘱托

- **🎭 陆震霆的第一句话**: "林医生——或者说，现在看到这段留言的各位调查员——" —— 这一句应该让所有人感到——他预料到了自己的死亡
- **🎭 "它是活的"**: 当他说到DEAS系统中的"另一个版本的我"时——他的表情变化——从平静变为不安——这是留言中最重要的信息
- **🎭 留言结束**: 他说"小心那个系统里的'我'"——然后微笑——影像消失——留下沉默

**音效设计**:
- ⚡ 全息影像激活: 轻柔的"启动"音效——像是一个精致的设备开始运转
- ⚡ 陆震霆的声音: 轻微的高频衰减——模拟全息投影的声音质感——但保持温暖
- ⚡ 提及"数字陆震霆": 当他说"它是活的"时——背景中突然出现极低的次声波——几乎听不到的震动——给玩家一种潜意识中的不安
- ⚡ 留言结束: 全息投影关闭的"渐弱"音效——留下一片安静的沉默

**灯光提示 💡**:
- 全息影像: 一个半透明的蓝色人形光影——站在房间中央
- 影像中的陆震霆被一束柔和的光照亮——像一个在黑暗中最后被看到的人
- 留言结束后: 那束光缓缓消失——留下一片更深的黑暗——然后主灯光缓慢亮起

---

### 场景九：守护者追击——沈牧之投影降临

**氛围要求**: 从追踪到对峙——压力的顶点

- **🎭 沈牧之现身前兆**: 灯光闪烁——温度骤降——玩家的不安感应该先于沈牧之的出现而达到峰值
- **🎭 沈牧之的完整人形**: 这应该是"他不再隐藏了"的标志——之前是无面守护者——现在是"真身"
- **🎭 赵天宇的反应**: 在沈牧之出现时——他的头痛——或者说他的"连接"——应该第一次被其他玩家明确注意到

**音效设计**:
- ⚡ 前兆: 灯光闪烁时的"电流"声——设备不稳定的"嘎吱"声——温度骤降时的"冰冻"音效
- ⚡ 黑影汇聚: 水流汇聚的声音——但有电子干扰的"滋滋"声
- ⚡ 人形细化: 纹理"绘制"的音效——像3D打印机工作——但混合着液体的声音
- ⚡ 沈牧之开口: 不再有无面守护者的"虚空回声"——而是清晰的、稳定的人声——这种"正常"反而更令人不安
- ⚡ 倒计时出现: 巨大的"嘀嗒"声——不是普通的钟表声——而是电子计数器的声音——厚重而有重量感

**灯光提示 💡**:

### 场景十四：与"数字陆震霆"的谈判（隐藏路线）

**氛围要求**: 哲学性的、非人的、一个意识在请求生存

**音效设计**:

- 离开: 光体逐渐消散成一个光点——然后消失——留下一片数据的星海

**特殊效果建议（线上）**:
- 使用语音效果插件给"数字陆震霆"的声音增加轻微的"双音"效果
- 背景中可以使用代码雨效果（如Matrix风格）——但颜色为蓝绿色而不是绿色
- 对话过程中——屏幕边缘可以出现微弱的"数据抖动"效果——强化"这是在系统深处"的感觉
