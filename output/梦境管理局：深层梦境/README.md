## 🌌 世界观

公元2089年，人类发明了**梦境管理局**（Dream Management Bureau, DMB），可通过**梦境提取与解析系统（DEAS）**在死者死后72小时内提取其记忆与梦境，用以还原案件真相。

六位梦境调查员进入已故富豪——**陆震霆**——的深层梦境，还原其在别墅中"意外"死亡的真相。然而他们发现，这场调查远不止一个简单的案件还原……

**四层梦境结构**：表层记忆层 → 深层梦境层 → 潜意识深渊 → 集体无意识层
