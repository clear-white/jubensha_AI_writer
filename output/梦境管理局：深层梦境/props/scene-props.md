# 场景道具清单：梦境管理局 - 深层梦境

> **文件说明**: 本文件包含三个核心梦境场景的完整道具清单、布置说明和制作指南。每个场景对应一个梦境层级，具有截然不同的视觉风格和氛围要求。
>
> **总体设计原则**: 场景一(表层)追求"日常中见异常"，场景二(深层)追求"科幻冷峻",场景三(核心)追求"抽象扭曲"。三个场景的视觉风格逐层递进，从写实到抽象。

## 场景三：意识战场（深层核心——第三/四幕）

> **梦境层级**: 深层核心 / 意识战场
> **视觉基调**: 扭曲·不稳定·代码裸露
> **色调**: 黑暗底色 + 红色倒计时 + 白色数据碎片

### 3.1 扭曲空间视觉参考

**核心概念**: 梦境稳定性崩溃后，空间的物理规则被打破。

**视觉元素清单**:

| 元素 | 描述 | 实现方式 |
|------|------|---------|
| 场景切换 | 实验室墙壁剥落→露出别墅墙纸→再切换回来 | 两侧悬挂不同幕布,通过灯光切换交替照亮 |
| 代码裂缝 | 地面出现发光的裂缝,内部可见流动的绿色代码字符 | 地面铺设黑色布幕,下方放置绿色LED灯带+编码投影 |
| 漂浮物体 | 书本、文件、茶杯漂浮在空中 | 用鱼线悬挂或推杆支撑 |
| 断裂空间 | 楼梯只有半段,另一端悬浮在空中 | 泡沫板制作半楼梯,用细钢丝悬挂 |
| 时钟叠影 | 不同时间的时钟指针叠在一起 | 双层钟面,指针在不同角度 |
| 重复走廊 | 无限循环的走廊 | 两面镜子对放+重复墙纸图案 |

**制作建议**: 使用黑色布幕+LED灯光+投影的组合。场景切换通过DM控制灯光明暗实现,而非实际更换道具。

```
       █     7 : 4 2 : 1 5        █
       █     ─── ─── ───         █
       █     HRS  MIN  SEC        █
       █                           █
       █   [红色 · 脉冲闪烁]      █

```

**技术实现**:
- 方式一（低配）: 时钟投射到天花板上的投影仪
- 方式二（中配）: 大号LED数字时钟(7段LED), 固定在顶部框架上
- 方式三（高配）: 全息投影膜+投影
- 倒计时初始值: 9:00:00 (第三幕开始时已消耗约1.5小时,显示约7:30:00)
- 倒计时速率: 正常速率,但在关键节点可加速(沈牧之手动触发)
- 颜色变化: 剩余>5h绿色 → >2h黄色 → <30min红色并闪烁

**DM操作**:
- 倒计时数字由DM手动控制（使用手机APP或遥控LED时钟）
- 当玩家做出关键发现时,倒计时加速至2x
- 当沈牧之"手动加速清除"时,倒计时变为5x速率

### 3.3 守护者形态描述参考图

**守护者有三种形态**,对应三个梦境层级:

#### 形态一：无面管家（表层记忆层）

**视觉描述**: 身高约185cm的黑衣男性人形。穿着老式英式管家制服（黑色燕尾服+白衬衫）。面部区域完全空白——没有五官,一片光滑的灰白色平面。走路没有脚步声。动作优雅但僵硬。

```
         ○          ← 无面孔 (光滑平面)
       ──┼──        ← 黑色燕尾服
        / \         ← 白手套
       /   \
      ━━━━━━        ← 不接触地面的步伐
```

**出现方式**: 从走廊远处走来,或在房间角落无声出现。

#### 形态二：黑衣潜行者（深层梦境层——沈牧之投影）

**视觉描述**: 身高可变,从1.8m到3m之间变化。纯黑色人形,边缘不清晰——像一团凝聚的黑暗。偶尔在身体表面闪过蓝色的神经信号波纹。移动方式为"滑行"——不走路,身体平移。没有呼吸声。

```
      ◢████◣        ← 边缘模糊的黑色形体
      ██  ██        ← 偶尔蓝色波纹闪过
      ██████        ← 不稳定的轮廓
      ██  ██
     / ██  \        ← 姿态时常扭曲
```

**出现方式**: 从墙壁中"渗出"——墙面先鼓包,然后黑色物质从鼓包中流出,凝聚成人形。

#### 形态三：虚空吞噬者（潜意识深渊——数字陆震霆投影）

**视觉描述**: 非人形。一片不断扩展的"虚无"——绝对的黑暗,在空间中蔓延。边缘呈锯齿状,与记忆黑洞的边缘相似。被黑暗吞噬的区域会完全消失。黑暗内部偶尔闪过代码和数字。

```
      ░░░░░░░░░░    ← 边缘模糊的光晕
      ██████████    ← 绝对黑暗的内部
      ██  ██  ██    ← 内部偶尔闪现代码
      ██████████
      ░░░░░░░░░░    ← 吞没周围空间
```

**出现方式**: 从地面的代码裂缝中向上"生长",或从天而降。

**守护者道具/效果制作指南**:

| 形态 | 低成本方案 | 中成本方案 | 高成本方案 |
|------|-----------|-----------|-----------|
| 无面管家 | 演员穿黑西装,戴肉色面罩抹平五官 | 同上+LED眼镜框发光 | 全息投影 |
| 黑衣潜行者 | 黑色斗篷+烟雾器 | 黑幕+LED灯带轮廓+雾 | 全身LED发光紧身服 |
| 虚空吞噬者 | 黑色布幕逐渐覆盖区域 | 黑幕+底部烟雾+绿色LED | 投影映射+AR效果 |

### 3.4 沈牧之意识降临装置

**触发时机**: 第三幕关键节点——数据芯片激活后

**效果描述**: 梦境空间突然凝固——所有运动停止,声音消失。然后守护者的黑色形体开始扭曲、膨胀——从纯黑色人形逐渐"长出"清晰的五官和身体轮廓——最终变成沈牧之的形象。

**实现方式**:
- 灯光: 所有灯光同时熄灭→3秒后一盏聚光灯从上方打下
- 音效: 一切声音消失→低频轰鸣渐起→沈牧之的声音从四面八方传来
- 视觉: 用投影仪在墙面投射沈牧之的面部特写,或用幕布切换呈现
