## 六、打印和装订说明

### 卡片打印

```
【打印设置】
- 文件: evidence-cards.md 中的卡片设计
- 纸张: 按每张卡片的规格单独选纸
- 尺寸: 63x88mm (标准扑克牌)
- 排版: 每张A4纸排8张(4x2),留出血位3mm
- 裁切: 裁切后圆角处理(R3)

【打印步骤】
1. 将卡片设计导出为PNG/PDF(300dpi)
2. 按纸型分类打印
3. 裁切→圆角→按需做旧/特效
4. 分组装袋(按发放幕次)
```

### 剧本装订

```
【装订规格】
- 尺寸: A5 (148x210mm)
- 装订: 骑马订/胶装
- 封面: 200g铜版纸,覆哑膜
- 内页: 80g道林纸,双面打印
- 页数: 约20-30页/本(视角色而定)

【装订步骤】
1. 按角色导出剧本文本
2. 排版:A5分栏,字号11pt
3. 打印→装订→切边
4. 封面标注: 剧本名+角色名+"仅供本人阅读"
```

### DM指南打印

```
【规格】
- 尺寸: A4活页
- 装订: 活页夹/线圈装
- 封面: 透明封套+红色封面纸
- 内容: 完整DM指南 + 速查表 + 附录
```
