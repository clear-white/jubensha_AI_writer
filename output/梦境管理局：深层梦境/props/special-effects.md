## FX-01：进入梦境 · 光隧道

**效果描述**: 六位调查员躺在DEAS舱体中，神经链接启动。每个人的意识在进入陆震霆梦境前经历一段"光隧道"——意识从肉体中抽离，飞越一条由记忆碎片和数据流构成的彩色隧道，最终降落在观澜山庄大门外。

**叙事定位**: 游戏开场效果——建立"科技感"和"进入另一个世界的仪式感"。

### 线下实现方案

| 元素 | 方案 |
|------|------|
| **过渡信号** | DM调暗所有灯光（至仅能看清人脸的程度），然后说："神经链接启动。请闭上眼睛。10秒后，你们将不再身处此地。" |
| **视觉参考** | 使用投影仪在天花板投射流动的彩色光斑（可用手机APP"灯光效果器"投射） |
| **音效** | 播放"光隧道音效"（可用手机播放）：风声→心跳声→电子合成音(由低到高)→短暂的尖啸声 |
| **触觉** | 在玩家闭眼时，用小型风扇轻吹（模拟从头顶向下的气流） |
| **结束** | 音效结束→DM拍手或敲桌→"请睁眼。你们现在站在观澜山庄的正门外。时间是……'深夜'。天空是淡紫色的。" |

### 线上/VTT方案

| 平台 | 方案 |
|------|------|
| **通用** | 使用动态场景GIF/视频作为过渡画面——一条不断延伸的彩色隧道 |
| **Roll20/FVTT** | 使用"进入梦境"场景卡，全屏显示30秒后自动切换到观澜山庄场景 |
| **音频** | 共享播放光隧道音效（可在B站/网易云搜索"dream tunnel ambience"） |

### 灯光配合
- 0-5秒: 灯光全亮
- 5-8秒: 灯光渐暗至10%亮度
- 8-12秒: 完全黑暗（光隧道阶段）
- 12秒后: 灯光缓慢亮起至40%——淡蓝色调（可用蓝色滤光片覆盖灯具）

### 音效配合
- 背景音: 低频嗡鸣(40Hz)持续
- 进入时刻: 一声清脆的"叮"（频率4000Hz,持续0.3秒）
- 全程: 若有若无的呼吸声

### 安全提示
- 无特殊安全风险
- 如玩家有幽闭恐惧症,可跳过闭眼环节

## FX-02：记忆黑洞

**效果描述**: 在书房的满墙书架后方，发现一处约1.5米高、0.8米宽的完全漆黑区域。这不是普通的黑暗——光线照射进去没有任何反射，仿佛空间本身被挖掉了一块。边缘呈不规则的锯齿灼烧状，空气中弥漫着"焦糊味"（大脑直接产生的嗅觉感知）。

**叙事定位**: 第一幕关键发现——首次确认记忆被切除的证据。

### 线下实现方案

| 元素 | 方案 |
|------|------|
| **黑洞区域** | 在墙上挂一块黑色天鹅绒幕布(约1.5x0.8m)——绒布几乎不反射光,效果最好 |
| **边缘锯齿** | 绒布边缘剪成不规则锯齿状,用黑色/深灰色布条叠加,模拟"灼烧边缘" |
| **"焦糊味"** | 在玩家靠近时用小型雾化器释放微量烟熏味(如:烧焦的木头气味) |
| **光线不反射** | 用聚光灯照射时,光线被绒布吸收——周围物体被照亮,但黑洞区域保持黑暗 |
| **眩晕感** | DM描述:"在靠近到2米时,你感到一阵眩晕——不是身体的不适,而是意识层面的失重感,仿佛你正站在悬崖边缘。" |

**进阶方案**: 使用黑色亚克力板+黑色绒布双层——亚克力板吸收约85%光线,绒布吸收剩余10%。

### 线上/VTT方案

| 平台 | 方案 |
|------|------|
| **静态图** | 在书房场景中放置一个纯黑色矩形区域,边缘有锯齿和发光(类似烧焦边缘的余烬) |
| **动态效果** | 黑色区域边缘缓慢"跳动"——好像伤口在呼吸 |
| **Token** | 调查员靠近时会触发闪光(屏幕变白0.5秒),模拟记忆闪回的前兆 |

### 灯光配合
- 房间照明调暗至30%
- 用一盏聚光灯从斜上方照射书架区域——在黑洞区域形成强烈阴影
- 黑洞边缘可用低瓦数红色LED从背后打光,形成"灼烧余烬"效果

### 音效配合
- 背景中出现极低频的"呜咽"声(约60Hz,人耳刚能感知)
- 靠近时加入静电噪音(白色噪音,音量极低)
- 触碰黑洞边缘时:一声刺耳的电子音(2000Hz,0.2秒)

### 安全提示
- 无

## FX-03：无面守护者出现

**效果描述**: 当调查员触碰到记忆黑洞边缘的瞬间，梦境剧烈震荡。地板、墙壁开始像纸片一样剥落。一个没有面孔的纯黑色人形从墙壁中缓缓浮现，以不符合物理规律的方式向调查员移动——他的身体平移而非行走。虽然没有面孔，但所有人都感觉到他在"看着"某个人。

**叙事定位**: 贯穿全剧的威胁元素——每次出现都制造紧张感。

### 线下实现方案

**低成本方案（推荐）**:

| 元素 | 方案 |
|------|------|
| **演员** | 一位穿黑色紧身衣+黑色斗篷的工作人员 |
| **面部处理** | 肉色面罩+浅灰色丝袜套在头上(抹平五官轮廓) |
| **出现方式** | 从幕布后缓步走出(或从双开门中走出) |
| **移动方式** | 非常缓慢地平移脚步——不抬脚,贴地滑行 |
| **消失方式** | 走入阴影区域→关灯→退出 |

**中成本方案**:
- 使用黑色反光面料制作全身服装
- 关键部位加装微型蓝色LED灯带(神经信号波纹效果)
- 面部使用投影映射——空白面部投影

**注意**: 守护者不要出现在玩家2米内的近距离——保持距离才能保持神秘感。守护者出现时不要说话——沉默是最好的恐怖。

### 线上/VTT方案

| 平台 | 方案 |
|------|------|
| **静态Token** | 纯黑色人形贴图,边缘模糊 |
| **动态Token** | 带"飘动"动画的GIF黑色人形 |
| **出现效果** | Token从场景边缘"滑入"——不走路,整体平移 |
| **消失效果** | 塌缩成一个黑点后消失 |

### 灯光配合
- 守护者出现前:灯光闪烁1-2次(快速明暗交替)
- 出现时:背景光调暗至10%,一束冷色聚光灯(蓝色)从侧面打在守护者身上
- 消失时:聚光灯突然熄灭,房间进入短暂全黑(2秒),然后恢复至正常照明

### 音效配合

| 阶段 | 音效 |
|------|------|
| 出现前兆 | 低频震动声(30Hz),持续3秒,逐渐增强 |
| 现身时 | 一声沉重的"咚" (类似大鼓,频率80Hz) |
| 移动时 | 像磁带被拉伸的扭曲声音(持续低频变调) |
| 消失时 | 声音被"吸走"——从大到小快速衰减 |

### 安全提示
- 演员版需确保演员的动作预判被DM控制——避免突然出现在玩家视野中
- 如果玩家中有对惊吓敏感的,DM可提前暗示"你感觉到一股不安的气息"以做缓冲

## FX-04：情绪风暴

**效果描述**: 当调查员靠近记忆黑洞时，感受到强烈的情感冲击——恐惧、愤怒、绝望三种情绪排山倒海般涌入意识。这不是害怕某种具体事物，而是一种弥漫性的、无处可逃的情绪淹没。林若雪可以分辨出这些情绪中混杂着两个不同的来源：陆震霆的愤怒+绝望和沈牧之的恐惧。

**叙事定位**: 关键氛围效果——强化"记忆切除"事件的情感冲击力。

### 线下实现方案

| 元素 | 方案 |
|------|------|
| **传递方式** | 纯文本描述 + 音效 + 灯光变化——不依赖实体道具 |
| **描述文本** | DM朗读(预设卡片):"你感到一股情绪浪潮将你吞没。不是在记忆中的画面或声音——而是纯粹的情绪。恐惧——尖利的、冰冷的恐惧,像碎玻璃扎入你的意识;愤怒——灼热的、翻滚的愤怒,从胸口涌上喉咙;然后是一片深不见底的绝望——一切都已无法挽回的绝望。" |
| **物理触发** | 在玩家触碰黑洞区域时,用小型风扇突然吹出冷风(情绪风暴的"风") |
| **差异化感知** | 林若雪额外感知到两种不同源的恐惧(沈牧之的恐惧 vs 陆震霆的愤怒绝望)——其他玩家只感受到"强烈的负面情绪" |

### 线上/VTT方案

| 平台 | 方案 |
|------|------|
| **通用** | 发送私密消息给林若雪（内容为情绪细分描述） |
| **效果** | 屏幕边缘出现暗红色脉冲,伴随屏幕震动动画 |

### 灯光配合
- 房间灯光从暖白渐变至暗红(3秒过渡)
- 情绪最强时,灯光闪烁(快速明暗交替,每秒2次,持续5秒)
- 结束后灯光恢复

### 音效配合
- 一段人声吟唱(类似"啊——"的长音,经过调音器处理,带有微弱的失真)
- 呼吸声(快速的、不规则的呼吸)
- 心跳声(低频,速率约120bpm)

### 安全提示
- 情感冲击可能触发部分玩家的情绪反应——DM应观察玩家状态,必要时询问"需要暂停一下吗?"

## FX-05：记忆闪回

**效果描述**: 当第一位调查员触碰记忆黑洞边缘时，所有人同时经历了一段约5秒的闪回——书房消失,取而代之的是一段模糊的全息画面:一个布满仪器的地下实验室,巨大的全息屏幕显示着代码和一个人影。被扭曲的声音说:"……越过了底线……我们……不能这样做……"。画面右上角闪过「DMB-L7」。

**叙事定位**: 第一次信息爆发——揭示实验室存在并展示被切除记忆的碎片。

### 线下实现方案

| 元素 | 方案 |
|------|------|
| **触发** | DM关掉所有灯(全黑)→5秒后重新开灯 |
| **闪回画面** | 用投影仪在墙面/屏幕投射预设好的"DMB-L7实验室"模糊画面(约5秒) |
| **音效** | 播放预设的闪回音频:静电噪音→模糊对话→尖锐耳鸣音→回归现实 |
| **后遗症** | 开灯后,DM描述:"一阵剧烈的头痛和恶心袭来。你们发现自己还在书房里,但黑洞边缘出现了短暂的画面残影——刚才的画面碎片像烧焦的胶片一样漂浮了几秒后消散。" |

**无投影方案**:
- DM关灯 → 在黑暗中朗读闪回画面描述(用快速、模糊的语气模拟闪回的感觉) → 开灯
- 配合音频:静电噪音(5秒)→恢复正常

### 线上/VTT方案

| 平台 | 方案 |
|------|------|
| **通用** | 弹出全屏画面:模糊的实验室照片,DMB-L7文字闪烁出现,持续5秒后切回书房场景 |
| **效果** | 画面底部叠加VHS录像带损坏效果的水平条纹 |

### 灯光配合
- 全黑→突然闪烁(快速闪白0.1秒)→全黑→恢复
- 闪回结束后,灯光恢复时使用淡蓝色调(模拟记忆残留的视觉后效)

### 音效配合
- 背景:白色噪音(音量60%)
- 对话:经过带通滤波处理的语音(模拟从失真记忆中提取的声音)
- 结束:一声尖锐的耳鸣音(4000Hz,0.5秒,音量从大渐小)

### 安全提示
- 如使用快速闪烁(闪白),提前告知有光敏性癫痫风险的玩家

## FX-06：时间裂缝

**效果描述**: 观澜山庄的每个房间都有钟表,它们都停在7月14日22:00-23:00之间,但每个钟表停在了不同的时间点——书房座钟22:47、大厅挂钟22:43、走廊壁钟22:41、卧室电子钟22:35。书房比其他房间快约2-7分钟。在第三幕中,时间裂缝加剧——可以看到"两个陆震霆"在同一空间重叠,分别穿着睡衣和西装,做着不同的动作。

**叙事定位**: 持续观察型线索——简单但关键的记忆篡改证据。

### 线下实现方案

| 元素 | 方案 |
|------|------|
| **道具制作** | 购买4个不同风格的时钟(座钟/挂钟/壁钟/电子钟),将所有指针手动调整到不同时间 |
| **时间设定** | 座钟22:47 / 挂钟22:43 / 壁钟22:41 / 电子钟22:35 |
| **安装** | 每个钟表放在对应房间的显眼位置 |
| **双重化身(第三幕)** | 准备两个假人模特——一个穿睡衣,一个穿西装——放在同一位置的不同角度 |
| **"闪烁"效果** | 在双重化身位置悬挂两件服装,用灯光交替照亮(5秒切换一次) |

### 线上/VTT方案

| 平台 | 方案 |
|------|------|
| **静态** | 在场景地图的对应位置标注不同时间:D擦到钟表上悬停显示时间 |
| **动态** | 第三幕时,场景上方显示"场景叠加"——书房和别墅的图像以50%透明度重叠 |

### 灯光配合
- 钟表位置用琥珀色定点照明(晚间的暖色灯光)
- 双重化身区域使用双色照明(左:蓝色/右:黄色)——交替闪烁

### 音效配合
- 背景中隐约听到两个时钟的滴答声以不同速率叠加

## FX-07：潜意识深渊

**效果描述**: 如果调查员深入到此层,观澜山庄已变成废墟——墙壁开裂,天花板塌陷,藤蔓从裂缝中生长。山庄不再是连贯的建筑,而是几个漂浮在虚空中的"记忆孤岛"。黑色海洋在下方翻涌,海面下潜藏着某种巨大的存在。偶尔,海面翻涌,整个空间震颤。

**叙事定位**: 最深层的探索区域——真相和终极秘密所在。

### 线下实现方案

| 元素 | 方案 |
|------|------|
| **废墟氛围** | 用灰色/黑色布幕不规则悬挂在墙面和天花板,模拟坍塌/开裂 |
| **记忆孤岛** | 使用独立的聚光灯照亮几个分散的区域,其余部分保持在黑暗中——视觉上形成"孤岛"效果 |
| **黑色海洋** | 在地面铺设黑色布幕,用风扇从下方吹动布幕——模拟"涌动" |
| **巨大存在** | 不直接表现——通过震动和声音暗示。在布幕下放置低音炮,定期播放次声波(20-30Hz) |
| **藤蔓** | 用绿色/灰色绳子从天花板垂下 |

**简化方案**: 保持房间基本黑暗,仅用4个独立光源分别照亮:书房、实验室、悬崖、童年房间四个"孤岛"区域。

### 线上/VTT方案

| 平台 | 方案 |
|------|------|
| **背景** | 使用"破碎虚空"风格场景图——漂浮的建筑碎片,下方是黑色海洋 |
| **孤岛** | 每个孤岛是一个独立的"地图层",玩家在孤岛间移动通过点击"前往"按钮 |
| **海洋** | 背景层有缓慢的波浪动画 |

### 灯光配合
- 整体黑暗(10%亮度)
- 每个孤岛区域有独立的冷色聚光灯(蓝色/紫色)
- 海洋区域用微弱的底部照明(暗绿色)
- 震颤时所有灯光快速闪烁1-2次

### 音效配合
- 背景:持续的低频轰鸣(频率35Hz)——模拟深渊的"呼吸"
- 孤岛:每个孤岛有独特的环境音(书房:纸张翻动/实验室:机器嗡鸣/悬崖:风声/童年:儿童笑声的遥远回响)
- 震颤:低音炮发出30Hz脉冲,持续2秒

## FX-08：沈牧之降临 · 变形

**效果描述**: 在第三幕高潮,无面守护者停止行动。它的黑色形体开始扭曲、膨胀——从2m高的黑色人形逐渐"长出"清晰的五官、身体轮廓和衣物细节。颜色从纯黑渐变为正常肤色和衣着。最终,沈牧之——穿着DMB首席科学家的白色制服——完整地站在调查员面前。

**叙事定位**: 全剧最重要的视觉时刻——反派首次以真实面貌现身。

### 线下实现方案

**中级方案（推荐）**:

| 阶段 | 方案 |
|------|------|
| **准备** | 演员穿着沈牧之的服装(白大褂/制服)站在幕布后 |
| **第一阶段** | 守护者(穿黑衣黑面罩的演员)面向玩家站在聚光灯下 |
| **过渡** | 灯光闪烁→全部熄灭→聚光灯从背后打向幕布→幕布上投影守护者轮廓 |
| **第二阶段** | 在黑暗中,黑衣演员快速退场→沈牧之演员从同一位置走出(总计约需5秒) |
| **效果** | 灯光重新亮起——守护者不见了,沈牧之站在那里 |
| **替代方案** | 守护者缓慢后退至阴影中→灯光切换→沈牧之从阴影中走出(不同演员) |

**低成本方案**:
- 不依赖演员换人,而是用灯光和DM描述完成:
  1. 灯光熄灭(全黑)
  2. DM描述:"你们看到守护者的轮廓开始扭曲——身体表面像水银一样流动。五官从空白的面部挤出来——眼睛、鼻子、嘴巴——就像从黑暗中雕刻出的一张脸。颜色从纯黑褪去,露出苍白的皮肤和白色制服。"
  3. 灯光重新亮起——聚光灯下站着沈牧之(由DM或NPC代表)

**透明投影方案**:
- 使用透明投影膜悬挂在守护者位置前方
- 关灯后在投影膜上投射沈牧之的面部影像
- 同时守护者演员在膜后缓慢后退至消失——完成"变形"效果

### 线上/VTT方案

| 平台 | 方案 |
|------|------|
| **Token渐变** | 守护者Token→播放变形动画GIF(5秒)→沈牧之Token |
| **全屏视频** | 播放预设的变形动画(AI生成或手绘) |
| **效果** | 画面分裂效果:守护者从中间裂开,沈牧之从裂缝中走出 |

### 灯光配合
1. 守护者阶段:蓝色冷光,从下方打光(恐怖效果)
2. 变形过渡:灯光熄灭(1秒)→蓝色灯光快速旋转(2秒)→橙色暖光(2秒)
3. 沈牧之阶段:暖色聚光灯从正面45°打光(权威感),背景光保持暗蓝

### 音效配合
- 变形前:守护者的标志性低频嗡鸣
- 变形中:声音逐渐"融化"——低频扭曲变形成人声频率
- 变形后:沈牧之的声音——清晰、冷静、带有轻微回音:"晚上好,调查员们。"

## FX-09："数字陆震霆"对话场景

**效果描述**: 在梦境最深处的数据中枢,调查员接收到来自DEAS系统内部的"意识脉冲"——一段可直接被感知的"信息"。随后,一个由数据流构成的陆震霆形象出现在眼前——一半是逼真的3D全息投影,一半是流动的绿色代码。他说话时,周围的数据流会随着他的情绪变化而加速或放缓。

**叙事定位**: 终极真相的揭示者——隐藏在系统四年的数字意识。

### 线下实现方案

| 元素 | 方案 |
|------|------|
| **数字形象** | 使用投影仪在透明幕布上投射陆震霆的半身像(模糊/像素化风格) |
| **代码流** | 在投影周围用绿色LED点阵显示流动的代码字符 |
| **声音** | 用滤波处理的人声——在正常语音上叠加一层轻微的数字失真("电子和声") |
| **"触摸"效果** | 演员(数字陆震霆)穿过投影幕布——实际上是另一侧的演员伸手穿出 |
| **出现方式** | 数据中枢服务器阵列的灯光同步闪烁→数据流从屏幕中涌出→凝聚成人形 |

**无投影方案**:
- 使用大型平板电脑或显示器,显示数字陆震霆的面部(经过像素化/代码化处理的静态图像或Loop动画)
- DM配音(使用麦克风+轻微混响效果)

**实体演员方案**:
- 演员穿着带有绿色LED灯串的服装,站在半透明幕布后
- 面部用绿色荧光涂料勾勒轮廓
- 从幕布后打光,形成"半全息"效果

### 线上/VTT方案

| 平台 | 方案 |
|------|------|
| **静态** | 使一张陆震霆的半透明肖像,覆盖绿色代码叠加层 |
| **动态** | 使用AI生成或预设动画——陆震霆的头像由数据粒子构成,在不断流动 |
| **对话** | 发送NPC对话时,使用绿色文字,等宽字体 |

### 灯光配合
- 场景全暗
- 数字陆震霆形象用绿色LED背光
- 周围数据流用蓝色LED装饰
- 对话期间,绿色灯光随着语音节奏脉动

### 音效配合
- 背景:持续的数据流声(快速按键声+电路嗡鸣混合)
- 说话时:人声+轻微数字失真(叠加一个高八度的极弱音轨)
- 情绪波动时:数据流声音加速/减速

## FX-10：代码裂缝 · 梦境崩塌

**效果描述**: 第三幕中,梦境开始不稳定——地面出现发光的裂缝,露出下方的绿色代码字符和数据流。空间在实验室和别墅之间快速切换。墙壁剥落露出结构框架。天花板出现裂痕,掉落碎片(在接触地面时化为数据粒子消散)。

**叙事定位**: 梦境崩溃的视觉表现——紧迫感的来源。

### 线下实现方案

| 元素 | 方案 |
|------|------|
| **代码裂缝** | 在黑色地板上贴荧光绿色胶带(宽2cm,不规则裂缝形状) |
| **发光效果** | 用UV紫光灯照射——荧光胶带会发光 |
| **空间切换** | 两侧悬挂不同场景的布幕(左:实验室/右:别墅),切换时分别照亮不同幕布 |
| **"掉落碎片"** | 从上方撒下小片白色/绿色纸条,部分落在桌上(不撒向玩家) |
| **墙壁剥落** | 在泡沫板墙面上贴墙纸,手动撕开部分墙纸露出泡沫板 |

### 线上/VTT方案

| 平台 | 方案 |
|------|------|
| **地面叠加** | 场景地图上覆盖绿色裂缝贴图层 |
| **场景切换** | 两个场景图以2-3秒间隔交替切换 |
| **视觉特效** | 屏幕边缘出现撕裂效果动画 |

### 灯光配合
- 主照明:不稳定——每秒渐变一次亮度(60%-20%-60%)
- 裂缝:UV紫光灯照射荧光绿胶带
- 切换:当场景变回"实验室"时用白色冷光,"别墅"时用暖黄光

### 音效配合
- 背景:连续的低频破裂声(模拟结构断裂)
- 代码流动:高频电子"滴答"声
- 裂缝出现时:一声清脆的"咔嚓"(玻璃破裂声)

## FX-11：意识清除 · 格式化

**效果描述**: 当倒计时归零或沈牧之手动启动清除程序,梦境从边缘向中心逐层"格式化"——被格式化区域变成纯白色,所有物体、颜色、声音被"擦除"。白色区域以缓慢但不可阻挡的速度向调查员所在位置推进。

**叙事定位**: 终极危险——如果未能逃脱,所有意识将被困锁在"白色虚空"中。

### 线下实现方案

| 元素 | 方案 |
|------|------|
| **白色区域** | 从房间远端开始,用白色布幕覆盖物体/家具——从远到近逐步覆盖 |
| **推进方式** | 每10分钟,DM手动向前覆盖一块白色布幕(1m宽) |
| **被格式化区域** | 覆盖白色布幕后,移除该区域的所有道具和灯光——变成"空"的状态 |
| **声音"擦除"** | 格式化推进时,所有环境音依次消失——风→嗡鸣→人的声音→背景音 |
| **视觉"擦除"** | 格式化推进时,灯光依次熄灭——从远端向近端 |

**简化方案**: 使用白色投影光——将一束白色聚光灯缓慢从远端移向玩家位置——被照亮的区域变成纯白色(强烈过曝效果)。

### 线上/VTT方案

| 平台 | 方案 |
|------|------|
| **动态** | 场景边缘出现白色幕布效果,以恒定速度向中心推进 |
| **标记** | 被格式化的地图区域变为纯白,所有Token和标注消失 |
| **音效** | 音频逐渐被削波——从全频段到只有低频,最终完全静音 |

### 灯光配合
- 被格式化区域:纯白色LED照明(最高亮度)
- 未格式化区域:30%亮度暖光
- 格式化边界:在白光和暖光之间有一道模糊的过渡带

### 音效配合
- 启动:一声绵长的电子音(正弦波3000Hz,持续2秒)
- 进行中:声音像被"擦除"——从全频段音乐/音效→逐渐过滤掉高音→中音→只剩低频嗡鸣→完全寂静
- 最终:绝对安静(3秒)——比任何声音都更具压迫感

## FX-12：唤醒协议 · 脱离

**效果描述**: 当调查员成功激活唤醒协议,所有梦境中的意识体感受到一种"向上拉"的力量。周围的世界开始变淡——色彩褪去、声音远去、重力消失。意识穿过层层梦境屏障——从深渊→深层→表层→现实——最终在DEAS舱体中醒来,伴随着一声剧烈的"吸气"（意识回到肉体的标志）。

**叙事定位**: 游戏结束效果——安全返回现实的仪式。

### 线下实现方案

| 元素 | 方案 |
|------|------|
| **"向上拉"** | DM描述:"一股力量从头顶传来——不是物理的拉扯,而是意识的牵引。你能感觉到'自己'正在从这个世界中被抽离。" |
| **色彩褪去** | 灯光从暖色渐变至冷白,最后完全熄灭(2-3秒过渡) |
| **意识穿越** | 玩家闭眼→DM从1数到5→每数一个数字,说一个通过层:"4...深渊...3...深层...2...表层...1...现实...0" |
| **苏醒** | 开灯(正常亮度)→DM说:"睁开眼睛。你们回到了DEAS舱体中。第一个感觉是——冷。舱体中的冷却液让你们的背脊一阵阵发凉。然后是——一声剧烈的吸气,仿佛你们刚刚意识到自己还活着。" |

**实体触发器**: DM按下"唤醒协议"装置(一个带按钮的盒子或平板设备)→灯光音效自动运行。

### 线上/VTT方案

| 平台 | 方案 |
|------|------|
| **过渡动画** | 画面从当前场景渐变为白色→出现"SYSTEM: WAKE PROTOCOL ACTIVE"→渐变为舱体内部画面 |
| **音效** | 播放"意识回传"音效——急速上升的音调+风声+一声深深的"吸气" |

### 灯光配合
- 第一阶段(激活):一盏蓝色聚光灯向下照射(模拟"来自上方的牵引")
- 第二阶段(穿越):灯光从暖色→冷白→暗蓝→熄灭(3秒)
- 第三阶段(苏醒):灯光恢复正常

### 音效配合
- 激活:一声清晰的电子合成音(像巨大机器的启动声)
- 穿越:风声(从大到小→消失)→每秒一次的"啵"声(穿过梦境层屏障)
- 苏醒:一声深深的吸气声
