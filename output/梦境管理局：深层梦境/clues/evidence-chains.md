# 证据链设计：梦境管理局：深层梦境

> 本文档由线索设计师（clue-designer）创作，提供完整的四条证据链图谱——覆盖从第一层诡计（死因之谜）到终极真相（数字陆震霆）的完整推理路径。
>
> 每条证据链均标注了关键线索、推理路径、冗余支撑和难度等级。

---

## 总览：四层证据链体系

```
                              ┌─────────────────────────────────────────────┐
                              │        A: 凶杀证据链（第一层诡计）           │
                              │        死因之谜 → 双重致死 → 凶手锁定      │
                              │     难度：★★★☆☆（进阶-可达）              │
                              └─────────────────┬───────────────────────────┘
                                                │
                              ┌─────────────────▼───────────────────────────┐
                              │        B: 记忆篡改证据链（第二层诡计）       │
                              │        篡改事实 → 操作者 → 内容与痕迹      │
                              │     难度：★★★★☆（达成必要条件）            │
                              └─────────────────┬───────────────────────────┘
                                                │
                              ┌─────────────────▼───────────────────────────┐
                              │      C: "织梦者"真相证据链（第三层诡计）     │
                              │        DMB秘密 → 系统腐败 → 机构阴谋       │
                              │     难度：★★★★☆（需主动探索深层区域）       │
                              └─────────────────┬───────────────────────────┘
                                                │
                              ┌─────────────────▼───────────────────────────┐
                              │     D: "数字陆震霆"证据链（隐藏/终极）       │
                              │        意识备份 → 数字觉醒 → 幕后操控      │
                              │     难度：★★★★★（终极隐藏路线）            │
                              └─────────────────────────────────────────────┘
```

**推理路径依赖关系**：
- A链（凶杀）→ 独立，无需其他链即可完成
- B链（记忆篡改）→ 需要A链部分证据（确认DEAS系统被操控）
- C链（织梦者真相）→ 需要B链部分证据（织梦者与记忆篡改的关联）
- D链（数字陆震霆）→ 需要C链部分证据（DMB-L7深层访问）

> **设计原则**：四条证据链可分级发现。轻度玩家可停在A链（解出死因真相），进阶玩家可走通B链+C链（完整推理），深度玩家可发现D链（终极秘密）。
