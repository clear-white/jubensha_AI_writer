## 六、关系网络特征总结

### 核心特征
1. **高密度互联**: 每个角色至少有7条直接关系链接（含NPC），信息流通路径充分
2. **多元关系类型**: 信任、怀疑、操控、保护、秘密同盟——关系光谱完整
3. **隐藏关系分布均匀**: 每个角色2-3条隐藏关系，无"清白旁观者"
4. **无孤立节点**: 即使是"新人"赵天宇也通过植入装置与核心反派有强连接
5. **内外联动**: 方晴作为"不进入梦境"的角色，提供了现实世界的数据视角，与梦境内部的角色形成信息互补

### 关键关系路径图（最短路径——任意两角色之间最多2步可达）

```
陈墨 ─── 赵天宇 ─── 沈牧之
  │         │
林若雪    方晴
  │         │
周明远 ─── 苏雨桐 ─── DMB局长
```

> 所有6名角色均通过不超过2次跳转即可连接至核心案件（陆震霆之死）和幕后主谋（沈牧之/DMB局长）。
