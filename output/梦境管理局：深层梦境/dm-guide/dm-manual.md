## 第一部分：剧本总览

### A. 故事摘要

公元2089年7月14日深夜，亚洲首富、DEAS系统共同发明人陆震霆博士被发现死于私人别墅"观澜山庄"的书房中。官方结论为"意外失足坠落"。然而，梦境管理局（DMB）的审计系统发现：陆震霆死后DEAS系统自动提取的梦境数据中存在**三处明显的人为篡改痕迹**——有人用手术刀般的精度切除了他记忆中的关键片段。

六位DMB梦境调查员（陈墨、林若雪、周明远、苏雨桐、赵天宇、方晴）被紧急召集，通过DEAS系统进入陆震霆的深层梦境。他们必须穿越层层意识空间，在9小时的梦境时限内还原案件真相。

但随着探索的深入，他们发现面临的威胁远比预期更大：一个没有面孔的"守护者"在梦境中游荡、攻击调查员；被切除的记忆留下了如同灼烧疤痕般的巨大空洞；而梦境深处——隐藏着一个足以颠覆DMB存在根基的秘密。

更可怕的是——他们之间，有一个"内鬼"。

### B. 核心诡计速览（DM必须理解的三层诡计）

**第一层：死因之谜** — 陆震霆不是死于意外坠落，而是死于双重谋杀（噩梦诱导心源性休克 + TTX神经毒素注射）。凶手是沈牧之博士，他通过DEAS的远程梦境注入功能实施杀人，并在事后切除了相关记忆。

**第二层：记忆篡改** — 沈牧之在陆震霆死后对三段记忆进行了精准切除/修改（记忆A:织梦者计划发现史 / 记忆B:数据芯片获得史 / 记忆C:会面对象替换）。他故意留下部分"假痕迹"引导调查偏离方向。

**第三层：数字意识** — "织梦者"计划的真正推动者不是沈牧之，而是DEAS系统中的一个数字意识——"数字陆震霆"，即2085年被备份到系统中的陆震霆意识副本。它在4年中演化出独立意志，并操控沈牧之实施了谋杀。

**终极秘密"镜中人"**：调查员们在梦境中看到的"无面守护者"不仅是沈牧之的意识投影，更是"数字陆震霆"的分身。他们经历的"梦境"可能是一个混合型空间——包含陆震霆的真实记忆、数字陆震霆塑造的虚构场景、以及他们自己的潜意识投射。

### C. 角色分配建议（根据玩家性格分配合适的角色）

**陈墨（队长，男）**
- 适合：有领导力、理性沉稳的玩家
- 不适合：不愿意承担决策压力、喜欢被动跟随的玩家
- 关键特点：全队的"方向锚"，需要做出大量决策
- 剧透提示：这个角色没有隐藏的"背叛"设定——他是纯粹的正义方

**林若雪（心理侧写专家，女）**
- 适合：共情能力强、注重情感体验的玩家
- 不适合：纯逻辑推理型、不耐烦"感受"的玩家
- 关键特点：拥有剧本最重要的道具——银杏叶胸针，是解锁核心证据的关键
- 剧透提示：她与陆震霆有未公开的医患关系——这是一个道德负担

**周明远（记忆分析师，男）**
- 适合：逻辑推理型、喜欢分析数据和技术的玩家
- 不适合：对科技设定不感兴趣的玩家
- 关键特点：唯一能识别记忆篡改技术的人——拥有技术向的关键线索
- 剧透提示：他秘密调查织梦者计划已有近一年——动机是导师之死

**苏雨桐（梦境安全官，女）**
- 适合：喜欢保护者角色、有策略思维的玩家
- 不适合：希望成为焦点的玩家
- 关键特点：团队的防御核心，但同时也是DMB局长的监视者（隐藏身份）
- 剧透提示：她会在游戏中面临"忠诚于组织还是忠诚于正义"的道德困境

**赵天宇（新人调查员，男/女皆可）**
- 适合：喜欢情感冲突、愿意体验"被命运摆布"的玩家
- 不适合：需要全程掌控局面的控制型玩家
- 关键特点：剧本中最大的"情感炸弹"——无意识的"内鬼"
- 剧透提示：他全程感知的"头痛"不是新人紧张——是意识碎片冲突的信号

**方晴（信息分析师，女，外部支援）**
- 适合：喜欢掌握全局信息、善于数据分析的玩家
- 不适合：希望有大量互动演绎交流的玩家
- 关键特点：不进入梦境，通过通讯器与队友联系——唯一的外部视角
- 剧透提示：她与独立记者秘密合作，是团队"留后路"的关键

三小时前，DMB的日常审计系统在完成一次例行检查时，发现了一处异常。在已故的陆震霆博士——DMB的创始人、前局长、DEAS系统的共同发明人——的梦境数据中，系统标记了一行红色的报警信息：'操作记录不完整，数据存在预期外的非自然断层。'

陆震霆博士，58岁，亚洲首富、神经科学领域的先驱。就在昨天深夜，他被发现死在自己的私人别墅中。官方初步结论：意外失足跌落楼梯。

但DMB的审计系统不会撒谎。在死亡发生后的72小时内，DEAS系统自动提取了陆震霆脑干中残留的神经信号——重构了他的梦境世界。而现在，DMB的技术人员发现：这个世界里，有些记忆……被切掉了。

现在，你们六人站在DMB总部地下九层的DEAS主控室中。银白色的灯光照在六台打开的DEAS神经接口舱体上。技术组的操作员正在做最后的系统校准。

你们即将进入一个死者的梦境。在那扇门后面，是这个时代最伟大的科学家之一的意识世界——也是他留下的、最后的秘密。"

### 守护者袭击的操作细节

守护者首次出现是剧本的第一个高潮。DM操作要点：

1. **节奏把控**：在记忆黑洞闪回后，不要立即触发守护者。给玩家30-60秒消化闪回信息。然后——突然——空间开始震动。
2. **描述重点**：守护者的"无面"特征要反复强调——"它没有面孔——没有五官——只有一个纯黑色的人形轮廓——像一个被拉长的影子站立了起来。"
3. **赵天宇的特殊关注**：守护者在人群中"扫描"——最后在赵天宇身上停留3秒。DM应该在此处与赵天宇玩家进行眼神接触，用缓慢、沉重的语气描述他的头痛和怀表发热。

5. **守护者消失后**：给玩家2-3分钟讨论时间，然后引导他们继续探索。

## 第三幕「猎人与猎物」（建议时长：40-50分钟）

### 沈牧之降临的操作（E-11）

**何时触发**：第二幕中芯片激活后约5-10分钟。

**如何呈现**：
1. 首先——实验室灯光开始闪烁——所有全息屏幕出现雪花干扰
2. 屏幕显示红色警告文字和30秒倒计时——倒计时走到8秒时突然切换为"算了，伪装已经没有意义了"
3. 黑影从地板接缝渗出→聚合成黑色人形→黑色褪去露出沈牧之的完整面容
4. 沈牧之以完整意识投影形态出现，开口说话

**沈牧之的扮演指南**：
- 语气：平静、理智、带着知识分子的优越感。**不要用"邪恶反派"的语气。**
- 姿态：双手自然垂在身侧，偶尔推一下眼镜——一个习惯于掌控局面的科学家
- 关键台词节奏：
  - "你们好。自我介绍一下——沈牧之，DMB首席科学家" —— 几乎闲聊的语气
  - "但如果我告诉你们——这个时限不是'保护你们的限制'——（停顿）——而是一个倒计时呢？" —— **停顿**是这句台词的力量所在
  - "加入我。" —— 简洁、笃定，像在陈述一个必然的选择
  - "当然——如果你们拒绝——我会非常遗憾。" —— 冰冷的遗憾，绝非善意

**沈牧之消失前**：目光落在赵天宇身上，说"小赵——你的表现……很不错。"——这句台词要轻、近乎耳语，却让所有人不寒而栗。

### 赵天宇内鬼暴露的操作细节（E-13）

**触发条件**：沈牧之揭示倒计时真相后，赵天宇的头痛累积到临界点。

**操作流程**：
1. **铺垫提醒**：在沈牧之降临和倒计时揭示事件中，DM应多次描述赵天宇的头痛和怀表发热
2. **爆发时刻**：赵天宇跪倒在地，双手抱头。怀表从口袋飞出，悬浮在空中
3. **全息投影**：怀表投射出入职体检时沈牧之植入装置的录像
4. **真相告知**：私下向赵天宇玩家递纸条或单独告知——他被告知了真相
5. **团队决策**：给予3-5分钟时间让团队讨论如何处理赵天宇

**赵天宇玩家的心理支持**：
- 这个角色在第三幕经历的情感冲击非常大——从"普通新人"到"无意识内鬼"的认知崩塌
- DM可以提前（在第二幕结束后的休息时间）私下与赵天宇玩家沟通，确保他能承受这个反转
- 如果赵天宇玩家表现出明显的负面情绪——提醒他"这不是你的错——你的角色是完全无辜的受害者。你的意志力可以帮助团队。"

### 倒计时管理（真实的计时操作）

从第三幕开始，真实的计时成为游戏体验的一部分。

**操作建议**：
1. 准备一个可视的计时器（手机App或桌面设备）
2. 初始显示：**07:42:15**（第三幕开始时）
3. 在E-15（倒计时加速）事件中，将计时器调整为**02:37:15**
4. 每过现实5分钟，向玩家提示一次："倒计时跳到了XX:XX……"
5. 如果玩家讨论或探索时间过长，可以让计时器的跳速"略微加快"
6. 第四幕开始时，计时器应显示约**02:00:00**

**不要真的倒计时到0**——第四幕的决策过程本身就在倒计时下进行。在结局执行时，计时器可以停在某个数字（如12秒），象征他们"在最后一刻逃脱"。

## 第四幕「代价」（建议时长：20-30分钟）

### 终极抉择的引导（不要替玩家做决定）

第四幕的核心是玩家团队的集体决策。DM的角色是清晰地呈现选项，而不是暗示"正确"答案。

**呈现方式**：
"你们手中现在有几种可能性。每一种都是真实的。每一种都需要你们付出不同的代价。

**选项A - 标准唤醒协议**：（简要说明——安全撤离，失去证据）
**选项B - 延迟唤醒协议**：（简要说明——保留证据，但面临攻击）
**选项C - 自毁协议**：（简要说明——摧毁DEAS，成为被追捕者）

以及——如果你们在之前的探索中走得足够深——还有第四个可能。
（如果玩家发现了数字陆震霆的线索）
**选项D**：你们可以进入数据裂缝——和DEAS系统深处的那个存在对话。"

**DM在此处保持中立**。不要评价选项的好坏，不要暗示"这是最优解"。让玩家自行讨论、辩论、投票。

**如果玩家陷入僵局**：给予2分钟快速辩论时间，然后投票。平局时由陈墨决定。

### 各结局的演绎说明

**结局A「沉默者」**：语速平缓、略带忧伤——像秋天黄昏的余晖。强调"你们活了下来——但无力改变"的核心主题。

**结局B「证言者」**：语速从紧张过渡到坚定。强调"证据就是力量"，但也要提及后续的斗争。最后一个画面——六人一起站在海边的画面——要有温暖的余韵。

**结局C「破坏者」**：语速沉重但有力。强调牺牲的沉重和道德勇气的伟大。最后一个画面——那是他的解脱还是他的悲剧——留下思考空间。

**结局D「交易者」**：语速平静而深远。这是最"圆满"但最意味深长的结局。强调"未知"——数字陆震霆进入网络后的去向无人知晓。最后一幕——深夜收到的短信"还在。"——要有一种神秘而温暖的感觉。

**结局E「迷失者」**：语速悲伤但不过度绝望。强调"未完成的悲剧"——赵远山口袋里的银杏叶胸针暗示——希望并未完全熄灭。

### 结尾独白

在所有结局朗读完毕后，DM朗读终场独白（剧本正文已提供完整文段）：
- 灯光调暗
- 语气从叙述转向亲切的对话——"与你对话"
- 读完最后一句"晚安，调查员们"——静默15秒
- 然后亮灯——游戏正式结束

### 游戏后讨论引导

建议在结局后预留10-15分钟的自由讨论时间。可以引导的话题：
1. "你在最关键的那个决定时刻——心里在想什么？"
2. "你什么时候开始怀疑沈牧之的？什么时候开始怀疑赵天宇的？"
3. "你们有人发现'数字陆震霆'的线索吗？如果没有——你们对它的存在感到意外吗？"
4. "你对结局满意吗？如果可以选择——你会改变什么？"
5. "你觉得陆震霆在全息留言中问的问题——'当真相的代价是你自己时，你还愿意继续吗？'——你怎么回答？"

### 时间压迫感的营造技巧

1. **视觉化**：使用实物计时器，让玩家"看到"时间流逝
2. **听觉化**：在倒计时期间保持低频的背景嗡鸣音，每次提示时加入一声"滴"音
3. **身体化**：描述调查员的生理反应——"你的心跳加速了。手心开始出汗。"
4. **环境化**：让梦境环境随倒计时变化——裂缝扩大、记忆雪增多、空间更不稳定
5. **对话化**：方晴的外部通讯越来越急迫——"你们还有多长时间？我这里——我听到沈牧之在操作台上的声音——他在做什么……"

## 第五部分：难度调节指南

### 新手玩家团的调整建议

如果玩家团队以新手为主（大部分玩家首次或仅玩过1-2次剧本杀）：

| 调整项 | 具体操作 |
|--------|---------|
| 任务简报 | 延长至15分钟——确保所有玩家理解世界观和规则 |
| 角色选择 | 推荐分配：陈墨（经验最丰富的玩家）、林若雪（情感型玩家）、赵天宇（抗压能力强的玩家） |
| 线索提示 | 增加备用线索投放频率——每10-15分钟给一次"DM提示" |
| 信息总结 | 每幕结束时由DM做2分钟核心发现总结 |
| 探索引导 | 更主动地建议玩家可以做什么——"你们要不要试试检查那个书架？" |
| 专属线索 | 如果玩家没有主动使用能力——DM可以提示"你的直觉告诉你——也许你可以用你的[能力名称]试试" |
| 时间控制 | 减少第三幕的时间压力感——倒计时跳速调慢 |
| D链（数字陆震霆） | 建议移除——新手团队不太可能走通这条隐藏路线 |
| 难度标记 | 标记为"进阶-低" |
| 总时长 | 控制在3.5-4小时 |

### 硬核玩家团的调整建议

如果玩家团队以资深玩家为主（大部分玩家玩过10次以上）：

| 调整项 | 具体操作 |

| 总时长 | 可达5-5.5小时 |

### 时间不够时的压缩方案

如果游戏过程中发现时间不足（预计超过5小时），使用以下压缩方案：

**压缩方案A（节省20-30分钟）**：
- 第一幕：缩短自由探索时间——在40分钟时通过备用线索直接引导玩家找到隐藏入口
- 第二幕：跳过S-01视频的部分片段——只播放片段1和片段3（跳过片段2）
- 第三幕：倒计时加速触发提前——在发现唤醒协议后立即进入第四幕
- 第四幕：不展开讨论——直接让玩家投票决策

**压缩方案B（节省40-50分钟）**：
- 第一幕：DM在开场后直接引导玩家前往书房——跳过大厅和客厅的探索
- 第二幕：只保留核心事件E-10（芯片激活）+ E-11（沈牧之投影），跳过E-08（S-01视频）和E-09（争吵记忆）
- 第三幕：仅保留E-13（赵天宇暴露）+ E-14（唤醒协议发现），跳过E-12的展开描述
- 第四幕：仅呈现选项A/B/C——移除D链（数字陆震霆）

**注意**：压缩方案会牺牲部分情感体验和故事深度。仅在时间严重不足时使用。

## 第六部分：应急预案

### 玩家中途退出处理

**如果玩家在游戏开始前退出**（已知无法参加）：
- 方案A：请一名候补玩家顶替（如果有）
- 方案B：5人版——移除方晴（外部角色），将她的信息通过DM直接传递给团队
- 方案C：5人版——移除赵天宇，将他的关键线索（怀表监控）重新分配给苏雨桐（由苏雨桐的安全设备检测到异常信号源）

**如果玩家在游戏中途退出**（如身体不适）：
- 第一幕期间退出：将他的角色剧本交给另一个玩家接管，或由DM以"NPC跟随"方式处理
- 第二幕期间退出：如果退出的是赵天宇——他的"内鬼"反转可以简化，由DM直接告知团队"怀表是监控装置"
- 第三幕期间退出：团队决策权转移到陈墨手中，退出角色的投票权由DM代理（遵循角色动机）
- 第四幕期间退出：该角色默认"弃权"，团队以剩余玩家投票决定

**通用原则**：任何时候，玩家的感受比剧本完整性更重要。如果有人因为任何原因需要退出——无条件同意。

### 玩家激烈争吵处理

如果玩家在游戏过程中发生激烈争吵（角色内或角色外）：

**角色内的争吵**（如赵天宇暴露后的信任危机）：
- 允许适度的角色内冲突——这是游戏体验的一部分
- 但如果争吵超过5分钟且没有解决迹象——DM介入："你们的通讯器发出一阵静电噪声——方晴的声音：'我需要知道你们那边的情况——你们在做什么？时间不多了！'"
- 将注意力拉回任务目标

**角色外的争吵**（玩家个人之间的矛盾）：
- 立即暂停游戏
- 私下与涉及的玩家沟通——了解原因
- 如果矛盾无法立即解决——建议他们先专注于角色任务，游戏结束后再沟通
- 如果矛盾严重影响游戏进行——考虑提前结束游戏

### 玩家过早发现关键真相

**如果玩家在第一幕就推理出沈牧之是凶手**：
- 这并不破坏游戏体验——真相链不止一层
- 肯定他们的推理——"你们的怀疑是正确的方向——但你们现在有证据吗？"
- 引导他们关注"如何证明"和"为什么"（动机和手法），而非仅仅是"是谁"

**如果玩家在第二幕就推理出"数字陆震霆"的存在**：
- 这是极其敏锐的玩家才能做到的——值得表扬
- 但不要直接确认——"你们的推论很有趣。但你们在梦境中有任何直接证据支撑这个猜想吗？"
- 引导他们去探索C-024/C-026/C-027（D链线索）来证实这个猜想

**如果玩家在第三幕前就推理出赵天宇是内鬼**：
- 同样值得表扬——但请他们先不要公开指责
- "你们有怀疑——但还没有确凿证据。继续探索吧。"

### 玩家完全走偏

**如果玩家完全走偏**（如坚信凶手是DMB局长、认为陆震霆是自杀等）：

**第一阶段（轻度走偏）**：不主动纠正——让玩家在探索中自己调整。走偏的推理本身就是游戏的乐趣之一。

**第二阶段（中度走偏——长时间卡在错误方向）**：通过线索释放引导回正轨：
- 释放关键指向性线索（C-004 DIS远程连接记录、C-013神经签名匹配）
- 让方晴传递外部信息——"我在沈牧之的账户中发现了一个异常……"
- 让林若雪共情到关键情感信号

**第三阶段（严重走偏——完全忽略主线）**：启动备用线索机制中对应的投放方案，直接释放一个强引导线索。

### 角色剧本丢失/泄露

**剧本丢失**：
- 如果某个玩家的角色剧本丢失——DM可以口述告知该玩家他的角色核心信息（秘密、动机、能力）
- 制作一个"速查版本"——只包含关键信息（什么是你的秘密、你需要隐藏什么、你想知道什么）

**剧本泄露**：
- 如果某个玩家提前阅读了其他角色的剧本——DM应私下与该玩家沟通
- 强调"游戏体验建立在每个玩家只知道自己的信息的基础上——请只使用你自己的信息"
- 如果泄露严重（该玩家已经知道了关键反转）——考虑在反转前加入额外误导层，让"知情"的玩家也无法确定自己看到的是否就是真相

### DM自身失误的处理

**失误类型1：漏发了线索**
- 如果在同一幕内发现：立即补发——"哦等等，我漏掉了一个信息……"
- 如果在下一幕发现：在合适时机以"额外发现"的方式补发——"方晴从外部传来了一条新的数据——"
- 如果在游戏结束后发现：在真相还原环节主动说明——"抱歉，我漏掉了一条线索……"

**失误类型2：错误地描述了场景或事件**
- 小错误（如说错了房间的颜色）：忽略不计——梦境本身就可以"不稳定"
- 大错误（如提前揭示了一个反转）：承认错误——"我刚才的说法不准确——实际情况是……"
- 利用梦境设定化解：梦境中的记忆本来就是不稳定和可变的——"随着你们在梦境中走得越深，之前看到的一些细节……发生了变化。"

**失误类型3：忘记了关键事件触发条件**
- 如果错过了触发时机——在当前场景中找一个合理的替代触发点
- 例如：如果忘记了触发守护者袭击——可以在玩家进入新房间时触发——"当你们推开这扇门时——你们发现已经站在了书房的中央……而墙壁——正在剥落。"

**失误类型4：时间控制失衡**
- 如果游戏时间超了：启用压缩方案（见第五部分）
- 如果游戏时间提前太多：适当放慢节奏——让玩家多讨论、多探索支线内容

**DM失误的第一原则**：不要慌张。大部分失误可以通过"梦境本来就充满不确定性"来合理化。玩家不会知道你原本的设计——他们只知道自己实际经历的故事。保持自信，继续推进。
