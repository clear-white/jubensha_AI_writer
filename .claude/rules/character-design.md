# 角色设计规则

## 核心原则

### 1. 人人有秘密
每个角色必须拥有至少1个核心秘密和2-3个次要秘密。
- 核心秘密：一旦曝光会改变局势
- 次要秘密：增加角色厚度和可探索性

### 2. 人人有动机
每个角色必须拥有多层动机：
- **表面动机**：可以公开谈论的目标
- **深层动机**：角色真正在意的事
- **隐藏动机**：与案件相关的秘密目的

### 3. 关系网络
角色间必须有足够的连接：
- 每个角色至少与其他3个角色有直接关系
- 关系类型必须多样化（友情/爱情/敌意/利益/血缘）
- 至少包含1-2个隐藏关系

### 4. 能力平衡
- 每个角色有独特的信息获取渠道
- 每个角色有独特的技能或知识
- 没有角色拥有压倒性优势
- 每个角色都有弱点和限制

### 5. 参与度保证
- 每个角色拥有至少1条关键线索
- 每个角色在每轮都有事情做
- 每个角色都有合理的怀疑对象
- 每个角色都有需要辩护/隐瞒的内容

## 角色档案模板

```yaml
角色名称:
  基本信息:
    年龄: 
    职业: 
    外貌特征:
    性格类型: (MBTI/九型人格)
  语言风格:
    说话的特色:
    口头禅:
    常用词汇:
  背景故事:
    成长经历:
    关键事件:
    当前状态:
  动机系统:
    表面动机:
    深层动机:
    隐藏动机:
  秘密:
    核心秘密:
    次要秘密:
  关系:
    角色A: 关系类型/亲密度/隐藏关系
    角色B: 关系类型/亲密度/隐藏关系
  关键信息:
    知道的信息:
    不知道的信息:
    需要发现的信息:
  技能:
    特殊能力:
    限制条件:
```

## 禁止事项

- ❌ 角色完全没有秘密
- ❌ 角色对案件毫无影响
- ❌ 角色的关键信息可以被其他角色完全替代
- ❌ 角色的行为动机不合理
- ❌ 角色的信息量与其他角色严重不平衡
