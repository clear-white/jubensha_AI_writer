# Post-Tool Hook

## 用途
在工具调用执行后执行的逻辑。

## 检查清单

### 文件写入后
- [ ] 更新记忆（如果写入的是重要创作文件）
- [ ] 记录文件变更到项目日志
- [ ] 如果写入的是剧本文件，触发连续性检查（可选）

### Agent 调用后
- [ ] 记录 Agent 输出摘要
- [ ] 检查是否有错误需要处理
- [ ] 更新进度状态

### 工作流节点后
- [ ] 检查是否到达检查点
- [ ] 如果是关键节点，输出进度报告
- [ ] 检查是否需要用户确认

## 自动行为

1. 每次完成一个创作阶段后，自动更新 todo 列表
2. 检测到审核报告中的严重问题，自动暂停流程并通知用户
3. 完成重要里程碑（如剧本初稿完成），输出简短的进度总结
